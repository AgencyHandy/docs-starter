Die Registrierung bei AgencyHandy dauert nur wenige Minuten. Sie erstellen Ihr Konto, wählen einen einzigartigen Workspace-Namen, der zur Subdomain Ihrer Agentur wird, verifizieren Ihre E-Mail-Adresse und durchlaufen einen kurzen Onboarding-Prozess, damit die Plattform von Anfang an auf Ihre Agentur zugeschnitten ist.

<Note>
  Stellen Sie vor dem Start sicher, dass Sie Zugang zu dem E-Mail-Postfach haben, mit dem Sie sich registrieren — Sie benötigen es, um Ihr Konto während der Einrichtung zu verifizieren.
</Note>

## Ihr Konto erstellen

<Steps>
  <Step title="Zur Registrierungsseite navigieren">
    Öffnen Sie Ihren Browser und navigieren Sie zu [app.agencyhandy.com/sign-up](https://app.agencyhandy.com/sign-up). Klicken Sie auf der Startseite auf **Kostenlos starten**, um zum Registrierungsformular zu gelangen.
  </Step>

  <Step title="Ihre Registrierungsdaten eingeben">
    Füllen Sie alle Pflichtfelder im Registrierungsformular aus:

    | Feld | Was einzugeben ist |
    |---|---|
    | **E-Mail-Adresse** | Eine gültige E-Mail-Adresse, auf die Sie zur Verifizierung zugreifen können |
    | **Vorname** | Ihr Vorname |
    | **Nachname** | Ihr Familienname |
    | **Passwort** | Ein sicheres Passwort, das die nachstehenden Anforderungen erfüllt |
    | **Workspace-Name** | Eine eindeutige Kennung für Ihre Agentur (siehe Regeln unten) |

    **Passwortanforderungen:**
    - Mindestens 8 Zeichen lang
    - Enthält mindestens einen Großbuchstaben
    - Enthält mindestens einen Kleinbuchstaben
    - Enthält mindestens eine Zahl
    - Enthält mindestens ein Sonderzeichen (z. B. `!`, `@`, `#`)
    - Darf keine Leerzeichen enthalten

    **Regeln für den Workspace-Namen:**
    - Nur Kleinbuchstaben und Zahlen
    - Bindestriche (`-`) sind zwischen Zeichen erlaubt
    - Mindestens 3 Zeichen
    - Keine Leerzeichen oder Großbuchstaben

    <Tip>
      Wählen Sie Ihren Workspace-Namen sorgfältig — er wird Teil der Subdomain Ihrer Agentur und repräsentiert Ihre Marke. Ein Workspace mit dem Namen `bright-agency` bildet zum Beispiel die Grundlage Ihrer einzigartigen AgencyHandy-URL.
    </Tip>
  </Step>

  <Step title="Den Datenschutzrichtlinien zustimmen">
    Aktivieren Sie das Kontrollkästchen, um zu bestätigen, dass Sie AgencyHandys **Datenschutzrichtlinie** akzeptieren. Ohne diese Zustimmung können Sie nicht fortfahren.
  </Step>

  <Step title="Das Registrierungsformular absenden">
    Klicken Sie auf die Schaltfläche **Registrieren**. AgencyHandy erstellt Ihr Konto und sendet sofort eine Verifizierungs-E-Mail an die von Ihnen angegebene Adresse.
  </Step>

  <Step title="Ihre E-Mail-Adresse verifizieren">
    Öffnen Sie die Verifizierungs-E-Mail in Ihrem Postfach. Sie enthält einen einzigartigen Verifizierungscode. Kehren Sie zur AgencyHandy-Verifizierungsseite zurück, geben Sie den Code in das dafür vorgesehene Feld ein und klicken Sie auf **Weiter**.

    <Note>
      Wenn die E-Mail nicht innerhalb weniger Minuten erscheint, überprüfen Sie Ihren Spam- oder Junk-Ordner. Wenn der Verifizierungscode abgelaufen ist, klicken Sie auf der Verifizierungsseite auf **Code erneut senden**, um einen neuen zu erhalten.
    </Note>

    Sobald AgencyHandy den Code akzeptiert, wird Ihr Konto bestätigt und Sie sehen eine Erfolgsmeldung.
  </Step>

  <Step title="Ihre Agenturdetails einrichten">
    Nach der Verifizierung hilft ein kurzer Onboarding-Fragebogen AgencyHandy dabei, Ihr Erlebnis anzupassen:

    1. **Agenturtyp** — Wählen Sie die Kategorie, die Ihre Agentur am besten beschreibt (z. B. Marketing, Design, Entwicklung, Beratung).
    2. **Nutzungsweise** — Beantworten Sie eine kurze Frage darüber, wie Sie AgencyHandy einsetzen möchten.
    3. **Teamgröße** — Wählen Sie die Anzahl der Personen in Ihrem Team.
    4. **Empfehlungsquelle** — Wählen Sie aus, wie Sie von AgencyHandy erfahren haben.

    Klicken Sie nach jedem Schritt auf **Weiter**, um den Prozess fortzusetzen.
  </Step>

  <Step title="Auf Ihr Agentur-Dashboard gelangen">
    Nach Abschluss der Onboarding-Schritte leitet AgencyHandy Sie zu Ihrem **Unternehmens-Start-Dashboard** weiter. Ihr Workspace ist jetzt aktiv und einsatzbereit.
  </Step>
</Steps>

## Mit Google registrieren

Wenn Sie kein separates Passwort erstellen möchten, können Sie sich stattdessen mit Ihrem Google-Konto registrieren.

<Steps>
  <Step title="Auf 'Mit Google fortfahren' klicken">
    Klicken Sie auf der Registrierungsseite auf die Schaltfläche **Mit Google fortfahren**.
  </Step>
  <Step title="Ihr Google-Konto auswählen">
    Wählen Sie das Google-Konto, das Sie mit AgencyHandy verknüpfen möchten, und erteilen Sie die angeforderten Berechtigungen.
  </Step>
  <Step title="Onboarding abschließen">
    AgencyHandy erstellt Ihr Konto mit Ihren Google-Profildaten. Sie werden dennoch durch die Onboarding-Schritte zu Agenturtyp und Teamgröße geführt, bevor Sie auf Ihrem Dashboard landen.
  </Step>
</Steps>

<Tip>
  Wenn Sie sich mit Google registrieren, verwenden Sie bei jeder Anmeldung die Schaltfläche **Mit Google fortfahren** — Ihre Google-E-Mail wird zu Ihrer primären Anmeldeidentität.
</Tip>

## Wichtige Hinweise

<Warning>
  Ihr Workspace-Name kann später nicht geändert werden, ohne den Support zu kontaktieren. Stellen Sie sicher, dass er Ihre Agentur korrekt repräsentiert, bevor Sie ihn absenden.
</Warning>

- Verwenden Sie ein starkes, einzigartiges Passwort und speichern Sie es in einem Passwort-Manager.
- Der an Ihre E-Mail gesendete Verifizierungscode kann nach kurzer Zeit ablaufen. Falls dies der Fall ist, fordern Sie mit **Code erneut senden** einen neuen an.
- Die Onboarding-Fragen schränken Sie nicht auf bestimmte Funktionen ein — sie personalisieren lediglich Ihre anfängliche Dashboard-Ansicht.
- Sobald Sie auf dem Dashboard sind, können Sie sofort damit beginnen, Kunden hinzuzufügen, Leistungen zu erstellen und Ihr erstes Angebot zu verfassen.
