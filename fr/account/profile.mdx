Votre profil AgencyHandy est l'identité que vos clients et coéquipiers voient sur toute la plateforme — des conversations de commande aux commentaires sur les fichiers. Le tenir exact et à jour rend la collaboration plus claire et garantit la sécurité du compte. Cette page vous guide pour mettre à jour vos informations personnelles, définir une photo de profil, changer votre mot de passe, personnaliser vos préférences de notification et utiliser un code promotionnel.

## Mettre à jour les informations de votre profil

<Steps>
  <Step title="Accédez aux paramètres du profil">
    Cliquez sur votre **icône de profil** dans le coin supérieur droit de l'interface, puis sélectionnez **Profile Settings** dans le menu déroulant.
  </Step>
  <Step title="Modifiez votre nom">
    Sur la page Profile Settings, cliquez sur le bouton **Edit** sur le côté droit. Cliquez dans le champ **Name** et saisissez votre nom mis à jour.
  </Step>
  <Step title="Enregistrez vos modifications">
    Cliquez sur **Save Changes** pour appliquer la mise à jour. Votre nouveau nom apparaît immédiatement sur toute la plateforme.
  </Step>
</Steps>

## Téléverser ou changer votre photo de profil

Une photo de profil aide les clients et les coéquipiers à vous identifier en un coup d'œil dans les conversations et les commentaires de tâches.

<Steps>
  <Step title="Ouvrez la section photo de profil">
    Dans Profile Settings, cliquez sur la zone **Update Profile Picture** (l'avatar actuel ou le cercle d'emplacement réservé).
  </Step>
  <Step title="Sélectionnez une nouvelle image">
    Cliquez sur **Upload New Picture** et choisissez une image depuis votre appareil. Formats pris en charge : **PNG, JPG, JPEG**. Taille maximale du fichier : **5 Mo**.
  </Step>
  <Step title="Ajustez et enregistrez">
    Recadrez ou ajustez l'image au besoin dans l'éditeur, puis cliquez sur **Save**. Votre nouvel avatar est appliqué immédiatement.
  </Step>
</Steps>

<Note>
  Si votre photo de profil ne se met pas à jour, vérifiez que le fichier fait moins de 5 Mo et qu'il est au format PNG, JPG ou JPEG.
</Note>

## Changer votre mot de passe

Mettre à jour régulièrement votre mot de passe aide à garder votre compte sécurisé.

<Steps>
  <Step title="Accédez à la section Password">
    Dans Profile Settings, faites défiler jusqu'à la section **Password**.
  </Step>
  <Step title="Saisissez votre mot de passe actuel">
    Saisissez votre mot de passe existant dans le champ **Current Password**. Cela confirme que c'est bien vous qui effectuez le changement.
  </Step>
  <Step title="Définissez votre nouveau mot de passe">
    Saisissez un nouveau mot de passe dans le champ **New Password**. Votre mot de passe doit inclure tous les éléments suivants :

    - Au moins un **chiffre**
    - Au moins une **lettre majuscule**
    - Au moins une **lettre minuscule**
    - Au moins un **caractère spécial** (par ex. `!`, `@`, `#`)
  </Step>
  <Step title="Confirmez et appliquez">
    Saisissez à nouveau le nouveau mot de passe dans le champ **Confirm Password**, puis cliquez sur **Update Password**.
  </Step>
</Steps>

<Warning>
  Si vous oubliez votre mot de passe actuel, utilisez le lien **Forgot Password** sur la page de connexion pour le réinitialiser par e-mail. Vous ne pouvez pas mettre à jour votre mot de passe depuis Profile Settings sans connaître l'actuel.
</Warning>

## Configurer les paramètres de notification

AgencyHandy vous permet de choisir quels événements déclenchent une notification et si ces notifications sont livrées **dans l'application**, **par e-mail** ou les deux. Chaque utilisateur gère ses propres préférences de manière indépendante.

<Steps>
  <Step title="Ouvrez Notification Settings">
    Dans la barre latérale, cliquez sur **Settings**, puis sélectionnez l'onglet **Notification Settings** en haut de la page.
  </Step>
  <Step title="Parcourez les catégories de notifications">
    Les notifications sont regroupées par type d'événement. Vous verrez des sections pour :

    - **Task & Ticket** — par ex., lorsqu'un client crée un nouveau ticket d'assistance
    - **Orders** — par ex., lorsqu'un client passe une nouvelle commande
    - **Payments** — par ex., lorsqu'un paiement est traité avec succès
    - **Comments** — par ex., lorsqu'une personne ajoute un commentaire ou un retour sur un fichier
  </Step>
  <Step title="Activez ou désactivez les canaux">
    Pour chaque événement, cochez ou décochez les cases **In App** et **Mail** pour contrôler la manière dont vous recevez cette alerte.

    Survolez l'icône **ⓘ** à côté du nom de chaque événement pour lire une description du moment précis où cette notification se déclenche.
  </Step>
  <Step title="Enregistrez vos préférences">
    Les modifications sont enregistrées automatiquement à mesure que vous cochez ou décochez chaque case — pas besoin de cliquer sur un bouton Enregistrer distinct.
  </Step>
</Steps>

### Référence des événements de notification

| Événement | Dans l'application | E-mail | Description |
|---|---|---|---|
| Ticket Created | ✅ Par défaut | ✅ Par défaut | Un client ouvre un nouveau ticket d'assistance |
| New Order | ✅ Par défaut | ✅ Par défaut | Un client passe une nouvelle commande |
| Agency Payment Successful | ✅ Par défaut | ✅ Par défaut | Un paiement est traité avec succès |
| Comment Added | ✅ Par défaut | ✅ Par défaut | Un nouveau commentaire est publié sur une tâche, un ticket ou un fichier |
| File Feedback Comment Added | ✅ Par défaut | ✅ Par défaut | Un client ou un coéquipier laisse un retour sur un fichier partagé |

<Tip>
  Si votre boîte de réception devient encombrée, désactivez les notifications **Mail** pour les événements de moindre priorité comme les commentaires et conservez les alertes e-mail uniquement pour les événements à fort impact comme les nouvelles commandes et les paiements.
</Tip>

### Réinitialiser aux valeurs par défaut

Si vous souhaitez restaurer toutes les préférences de notification à la configuration d'origine de l'espace de travail, faites défiler jusqu'au bas de la page Notification Settings et cliquez sur **Reset to Default**.

<Note>
  Les préférences de notification sont personnelles — réinitialiser vos préférences n'affecte les paramètres d'aucun autre utilisateur. Seuls les Super Admins peuvent réinitialiser les valeurs par défaut globales de l'espace de travail.
</Note>

## Utiliser un code promotionnel

Si vous avez reçu un code promotionnel ou de réduction d'AgencyHandy, vous pouvez l'utiliser directement depuis les paramètres de votre compte.

<Steps>
  <Step title="Accédez aux paramètres">
    Cliquez sur **Settings** dans la barre latérale de gauche.
  </Step>
  <Step title="Ouvrez la section Redeem Promotion Code">
    Recherchez l'option **Redeem Promotion Code** dans les paramètres et cliquez dessus.
  </Step>
  <Step title="Saisissez votre code">
    Saisissez ou collez votre code promotionnel dans le champ de saisie et cliquez sur **Redeem**.
  </Step>
  <Step title="Confirmez la réduction">
    AgencyHandy valide le code et applique la réduction à votre compte. Un message de confirmation indique l'avantage qui a été appliqué.
  </Step>
</Steps>
