Profil AgencyHandy Anda adalah identitas yang dilihat klien dan rekan tim di seluruh platform — mulai dari percakapan pesanan hingga komentar file. Menjaganya akurat dan terkini membuat kolaborasi lebih jelas dan memastikan keamanan akun. Halaman ini memandu Anda memperbarui detail pribadi, menetapkan foto profil, mengubah kata sandi, menyesuaikan preferensi notifikasi, dan menukarkan kode promosi.

## Perbarui informasi profil Anda

<Steps>
  <Step title="Buka Pengaturan Profil">
    Klik **ikon profil** di sudut kanan atas antarmuka, lalu pilih **Profile Settings** dari menu dropdown.
  </Step>
  <Step title="Edit nama Anda">
    Di halaman Pengaturan Profil, klik tombol **Edit** di sisi kanan. Klik kolom **Name** dan ketik nama Anda yang diperbarui.
  </Step>
  <Step title="Simpan perubahan Anda">
    Klik **Save Changes** untuk menerapkan pembaruan. Nama baru Anda langsung muncul di seluruh platform.
  </Step>
</Steps>

## Unggah atau ubah foto profil Anda

Foto profil membantu klien dan rekan tim mengidentifikasi Anda sekilas dalam percakapan dan komentar tugas.

<Steps>
  <Step title="Buka bagian foto profil">
    Di Pengaturan Profil, klik area **Update Profile Picture** (avatar saat ini atau lingkaran placeholder).
  </Step>
  <Step title="Pilih gambar baru">
    Klik **Upload New Picture** dan pilih gambar dari perangkat Anda. Format yang didukung: **PNG, JPG, JPEG**. Ukuran file maksimum: **5 MB**.
  </Step>
  <Step title="Sesuaikan dan simpan">
    Potong atau sesuaikan gambar sesuai kebutuhan di editor, lalu klik **Save**. Avatar baru Anda langsung diterapkan.
  </Step>
</Steps>

<Note>
  Jika foto profil Anda tidak diperbarui, pastikan file berukuran di bawah 5 MB dan dalam format PNG, JPG, atau JPEG.
</Note>

## Ubah kata sandi Anda

Memperbarui kata sandi secara rutin membantu menjaga keamanan akun Anda.

<Steps>
  <Step title="Navigasi ke bagian Password">
    Di Pengaturan Profil, gulir ke bawah ke bagian **Password**.
  </Step>
  <Step title="Masukkan kata sandi saat ini">
    Ketik kata sandi yang ada di kolom **Current Password**. Ini mengonfirmasi bahwa Anda yang melakukan perubahan.
  </Step>
  <Step title="Tetapkan kata sandi baru">
    Masukkan kata sandi baru di kolom **New Password**. Kata sandi Anda harus mencakup semua hal berikut:

    - Setidaknya satu **angka**
    - Setidaknya satu **huruf besar**
    - Setidaknya satu **huruf kecil**
    - Setidaknya satu **karakter khusus** (misalnya `!`, `@`, `#`)
  </Step>
  <Step title="Konfirmasi dan terapkan">
    Masukkan kembali kata sandi baru di kolom **Confirm Password**, lalu klik **Update Password**.
  </Step>
</Steps>

<Warning>
  Jika Anda lupa kata sandi saat ini, gunakan tautan **Forgot Password** di halaman login untuk meresetnya melalui email. Anda tidak dapat memperbarui kata sandi dari Pengaturan Profil tanpa mengetahui kata sandi saat ini.
</Warning>

## Konfigurasikan pengaturan notifikasi

AgencyHandy memungkinkan Anda memilih peristiwa mana yang memicu notifikasi dan apakah notifikasi tersebut disampaikan **dalam aplikasi**, **melalui email**, atau keduanya. Setiap pengguna mengelola preferensi mereka sendiri secara independen.

<Steps>
  <Step title="Buka Pengaturan Notifikasi">
    Di bilah sisi, klik **Settings**, lalu pilih tab **Notification Settings** di bagian atas halaman.
  </Step>
  <Step title="Telusuri kategori notifikasi">
    Notifikasi dikelompokkan berdasarkan jenis peristiwa. Anda akan melihat bagian untuk:

    - **Task & Ticket** — misalnya, saat klien membuat tiket dukungan baru
    - **Orders** — misalnya, saat klien melakukan pesanan baru
    - **Payments** — misalnya, saat pembayaran berhasil diproses
    - **Comments** — misalnya, saat seseorang menambahkan komentar atau umpan balik file
  </Step>
  <Step title="Aktifkan atau nonaktifkan saluran">
    Untuk setiap peristiwa, centang atau hapus centang pada kotak **In App** dan **Mail** untuk mengontrol cara Anda menerima peringatan tersebut.

    Arahkan kursor ke ikon **ⓘ** di sebelah nama peristiwa apa pun untuk membaca deskripsi kapan tepatnya notifikasi tersebut diaktifkan.
  </Step>
  <Step title="Simpan preferensi Anda">
    Perubahan disimpan secara otomatis saat Anda mencentang atau menghapus centang setiap kotak — tidak perlu mengklik tombol Simpan terpisah.
  </Step>
</Steps>

### Referensi peristiwa notifikasi

| Peristiwa | Dalam Aplikasi | Email | Keterangan |
|---|---|---|---|
| Ticket Created | ✅ Default | ✅ Default | Klien membuka tiket dukungan baru |
| New Order | ✅ Default | ✅ Default | Klien melakukan pesanan baru |
| Agency Payment Successful | ✅ Default | ✅ Default | Pembayaran berhasil diproses |
| Comment Added | ✅ Default | ✅ Default | Komentar baru diposting pada tugas, tiket, atau file |
| File Feedback Comment Added | ✅ Default | ✅ Default | Klien atau rekan tim meninggalkan umpan balik pada file yang dibagikan |

<Tip>
  Jika kotak masuk Anda penuh, nonaktifkan notifikasi **Mail** untuk peristiwa prioritas rendah seperti komentar dan pertahankan peringatan email hanya untuk peristiwa berdampak tinggi seperti pesanan baru dan pembayaran.
</Tip>

### Reset ke default

Jika Anda ingin memulihkan semua preferensi notifikasi ke konfigurasi workspace asal, gulir ke bagian bawah halaman Pengaturan Notifikasi dan klik **Reset to Default**.

<Note>
  Preferensi notifikasi bersifat pribadi — mereset preferensi Anda tidak memengaruhi pengaturan pengguna lain. Hanya Super Admin yang dapat mereset default workspace global.
</Note>

## Tukarkan kode promosi

Jika Anda menerima kode promosi atau diskon dari AgencyHandy, Anda dapat menukarkannya langsung dari pengaturan akun Anda.

<Steps>
  <Step title="Navigasi ke Settings">
    Klik **Settings** di bilah sisi kiri.
  </Step>
  <Step title="Buka bagian Redeem Promotion Code">
    Cari opsi **Redeem Promotion Code** di dalam Settings dan klik.
  </Step>
  <Step title="Masukkan kode Anda">
    Ketik atau tempel kode promosi Anda ke dalam kolom input dan klik **Redeem**.
  </Step>
  <Step title="Konfirmasi diskon">
    AgencyHandy memvalidasi kode dan menerapkan diskon ke akun Anda. Pesan konfirmasi menampilkan manfaat yang telah diterapkan.
  </Step>
</Steps>
