AgencyHandy memberi setiap pesanan dan tugas area penyimpanan file tersendiri, sehingga deliverable, materi referensi, dan umpan balik klien tidak pernah hilang di utas email. Anda mengunggah file langsung ke tugas yang bersangkutan, membagikannya kepada klien atau rekan tim, dan menerima umpan balik sebagai komentar inline — semuanya tanpa meninggalkan platform. File disimpan dengan riwayat yang jelas, membuat tinjauan versi dan persetujuan menjadi mudah.

## Format file yang didukung dan batasannya

AgencyHandy menerima berbagai jenis file untuk mencakup setiap deliverable yang diproduksi agensi Anda.

<CardGroup cols={3}>
  <Card title="Dokumen" icon="file-lines">
    `pdf` `doc` `docx` `odt` `txt` `ppt` `pptx` `xls` `xlsx` `csv`
  </Card>
  <Card title="Gambar & Grafis" icon="image">
    `jpg` `jpeg` `png` `gif` `webp` `svg` `ai`
  </Card>
  <Card title="Media & Arsip" icon="file-zipper">
    `mp4` `mp3` `wav` `zip` `rar`
  </Card>
</CardGroup>

<Note>
  Ukuran file maksimum per unggahan adalah **1 GB**. Total kapasitas penyimpanan Anda bergantung pada paket Anda: 100 GB (Freelancer), 2 TB (Team Starter), atau 10 TB (Business Pro).
</Note>

## Unggah file ke tugas pesanan

File dilampirkan di tingkat tugas, menjaga deliverable tetap terorganisir dan terbatas pada pekerjaan yang persis berkaitan.

<Steps>
  <Step title="Buka tugas">
    Navigasi ke pesanan yang relevan, lalu klik tugas yang ingin Anda lampirkan file. Modal detail tugas akan terbuka.
  </Step>
  <Step title="Temukan bagian Attachment">
    Gulir ke bawah di dalam modal tugas untuk menemukan bagian **Attachment**.
  </Step>
  <Step title="Unggah file Anda">
    Anda memiliki dua opsi:

    - **Klik Upload** — pemilih file terbuka sehingga Anda dapat menelusuri penyimpanan lokal dan memilih satu atau beberapa file.
    - **Seret dan lepas** — seret file dari desktop Anda langsung ke area Attachment.

    AgencyHandy mengunggah file dan menampilkannya di daftar lampiran setelah transfer selesai.
  </Step>
  <Step title="Bagikan dengan tim atau klien">
    Setelah diunggah, Anda dapat membagikan file dengan anggota tim tertentu atau klien Anda dengan mengatur izin akses dari dalam tugas. File yang dibagikan muncul di tampilan tugas penerima.
  </Step>
</Steps>

## Lihat file yang diunggah

Semua file yang dilampirkan ke tugas terdaftar di bagian **Attachment** pada modal tugas. Klik file mana saja untuk membuka pratinjau (untuk format yang didukung) atau mengunduhnya. File ditampilkan beserta nama, tanggal unggah, dan pengunggahnya, memberi Anda jejak audit yang jelas.

<Tip>
  Perlu menemukan file di beberapa pesanan dengan cepat? Gunakan bagian **Files** global di bilah sisi untuk menelusuri semua aset yang diunggah dalam tampilan terpusat, yang dapat difilter berdasarkan proyek atau tanggal unggah.
</Tip>

## Alur kerja umpan balik file

Fitur umpan balik file memungkinkan klien dan anggota tim meninggalkan komentar langsung pada file yang dibagikan, menjaga semua catatan revisi di satu tempat daripada tersebar di berbagai email.

<Steps>
  <Step title="Bagikan file untuk ditinjau">
    Di bagian Attachment tugas, bagikan file dengan klien Anda. Mereka menerima notifikasi (dalam aplikasi dan/atau email, berdasarkan pengaturan notifikasi mereka) bahwa file siap untuk ditinjau.
  </Step>
  <Step title="Klien meninjau dan berkomentar">
    Klien membuka file dari portal mereka. Mereka dapat melihat file dan menambahkan **komentar umpan balik** untuk menandai perubahan, mengajukan pertanyaan, atau memberikan persetujuan.
  </Step>
  <Step title="Tim menerima notifikasi umpan balik">
    Saat klien menambahkan komentar umpan balik, AgencyHandy mengirimkan notifikasi **File Feedback Comment Added** kepada anggota tim yang relevan. Mengklik notifikasi membawa Anda langsung ke file dan komentar tersebut.
  </Step>
  <Step title="Tindaklanjuti umpan balik">
    Tinjau komentar klien, buat perubahan yang diperlukan, dan unggah file yang direvisi ke tugas yang sama. Tambahkan komentar balasan untuk memberi tahu klien bahwa revisi sudah siap.
  </Step>
  <Step title="Klien menyetujui">
    Setelah klien puas, mereka dapat menandai file sebagai disetujui. Status persetujuan dicatat terhadap file untuk catatan Anda.
  </Step>
</Steps>

<Note>
  Semua komentar umpan balik file disimpan di dalam AgencyHandy dan dikaitkan dengan tugas dan file tertentu, sehingga Anda selalu memiliki riwayat revisi lengkap yang tersedia.
</Note>

## Hapus file

Untuk menghapus file dari tugas, buka modal tugas, arahkan kursor ke file di bagian Attachment, dan klik **ikon hapus**. Konfirmasi penghapusan saat diminta. Menghapus file akan menghapusnya secara permanen dari tugas — tidak dapat dipulihkan.

<Warning>
  Menghapus file juga menghapus komentar umpan balik yang terlampir padanya. Pastikan Anda tidak lagi membutuhkan komentar atau file tersebut sebelum mengonfirmasi penghapusan.
</Warning>

## Manfaat utama sekilas

| Kemampuan | Detail |
|---|---|
| **Penyimpanan terpusat** | Semua file tersimpan di dalam AgencyHandy, tanpa drive eksternal atau lampiran email |
| **Riwayat versi** | Beberapa file dapat berdampingan dalam satu tugas sehingga versi sebelumnya tetap dapat diakses |
| **Tinjauan kolaboratif** | Klien berkomentar langsung pada file — tanpa email bolak-balik |
| **Kontrol akses** | Hanya pengguna dengan izin yang tepat yang dapat melihat, mengunggah, atau menghapus file |
| **Dukungan format luas** | Lebih dari 20 jenis file diterima, hingga 1 GB per file |
