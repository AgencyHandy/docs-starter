AgencyHandy 프로필은 주문 대화에서 파일 댓글에 이르기까지 플랫폼 전반에서 클라이언트와 팀원이 볼 수 있는 신원입니다. 정확하고 최신 상태로 유지하면 협업이 더 명확해지고 계정 보안이 보장됩니다. 이 페이지에서는 개인 정보 업데이트, 프로필 사진 설정, 비밀번호 변경, 알림 환경설정 맞춤 설정, 프로모션 코드 사용 방법을 안내합니다.

## 프로필 정보 업데이트하기

<Steps>
  <Step title="프로필 설정으로 이동">
    인터페이스 오른쪽 상단 모서리의 **프로필 아이콘**을 클릭한 후 드롭다운 메뉴에서 **프로필 설정**을 선택합니다.
  </Step>
  <Step title="이름 수정">
    프로필 설정 페이지에서 오른쪽의 **수정** 버튼을 클릭합니다. **이름** 필드를 클릭하고 업데이트된 이름을 입력합니다.
  </Step>
  <Step title="변경 사항 저장">
    **변경 사항 저장**을 클릭하여 업데이트를 적용합니다. 새 이름이 플랫폼 전반에 즉시 반영됩니다.
  </Step>
</Steps>

## 프로필 사진 업로드 또는 변경하기

프로필 사진은 클라이언트와 팀원이 대화 및 작업 댓글에서 한눈에 당신을 식별하는 데 도움이 됩니다.

<Steps>
  <Step title="프로필 사진 섹션 열기">
    프로필 설정에서 **프로필 사진 업데이트** 영역(현재 아바타 또는 자리 표시자 원)을 클릭합니다.
  </Step>
  <Step title="새 이미지 선택">
    **새 사진 업로드**를 클릭하고 기기에서 이미지를 선택합니다. 지원 형식: **PNG, JPG, JPEG**. 최대 파일 크기: **5 MB**.
  </Step>
  <Step title="조정 및 저장">
    편집기에서 필요에 따라 이미지를 자르거나 조정한 후 **저장**을 클릭합니다. 새 아바타가 즉시 적용됩니다.
  </Step>
</Steps>

<Note>
  프로필 사진이 업데이트되지 않는 경우 파일이 5 MB 미만이고 PNG, JPG, JPEG 형식인지 확인하세요.
</Note>

## 비밀번호 변경하기

비밀번호를 정기적으로 업데이트하면 계정을 안전하게 보호할 수 있습니다.

<Steps>
  <Step title="비밀번호 섹션으로 이동">
    프로필 설정에서 아래로 스크롤하여 **비밀번호** 섹션으로 이동합니다.
  </Step>
  <Step title="현재 비밀번호 입력">
    **현재 비밀번호** 필드에 기존 비밀번호를 입력합니다. 이렇게 하면 본인이 변경하는 것임을 확인합니다.
  </Step>
  <Step title="새 비밀번호 설정">
    **새 비밀번호** 필드에 새 비밀번호를 입력합니다. 비밀번호에는 다음이 모두 포함되어야 합니다:

    - **숫자** 최소 1개
    - **대문자** 최소 1개
    - **소문자** 최소 1개
    - **특수 문자** 최소 1개 (예: `!`, `@`, `#`)
  </Step>
  <Step title="확인 및 적용">
    **비밀번호 확인** 필드에 새 비밀번호를 다시 입력한 후 **비밀번호 업데이트**를 클릭합니다.
  </Step>
</Steps>

<Warning>
  현재 비밀번호를 잊어버린 경우 로그인 페이지의 **비밀번호 찾기** 링크를 사용하여 이메일로 재설정하세요. 현재 비밀번호를 모르는 상태에서는 프로필 설정에서 비밀번호를 업데이트할 수 없습니다.
</Warning>

## 알림 설정 구성하기

AgencyHandy를 사용하면 어떤 이벤트가 알림을 트리거할지, 해당 알림이 **앱 내**, **이메일**, 또는 둘 다로 전달될지를 선택할 수 있습니다. 각 사용자는 자신의 환경설정을 독립적으로 관리합니다.

<Steps>
  <Step title="알림 설정 열기">
    사이드바에서 **설정**을 클릭한 다음 페이지 상단의 **알림 설정** 탭을 선택합니다.
  </Step>
  <Step title="알림 카테고리 탐색">
    알림은 이벤트 유형별로 그룹화됩니다. 다음 섹션을 볼 수 있습니다:

    - **작업 및 티켓** — 예: 클라이언트가 새 지원 티켓을 생성할 때
    - **주문** — 예: 클라이언트가 새 주문을 할 때
    - **결제** — 예: 결제가 성공적으로 처리될 때
    - **댓글** — 예: 누군가 댓글이나 파일 피드백을 추가할 때
  </Step>
  <Step title="채널 활성화 또는 비활성화">
    각 이벤트에 대해 **앱 내** 및 **메일** 체크박스를 선택하거나 해제하여 해당 알림 수신 방법을 제어합니다.

    이벤트 이름 옆의 **ⓘ** 아이콘 위에 마우스를 올리면 해당 알림이 언제 발생하는지에 대한 설명을 읽을 수 있습니다.
  </Step>
  <Step title="환경설정 저장">
    각 체크박스를 선택하거나 해제하면 변경 사항이 자동으로 저장됩니다 — 별도의 저장 버튼을 클릭할 필요가 없습니다.
  </Step>
</Steps>

### 알림 이벤트 참조

| 이벤트 | 앱 내 | 이메일 | 설명 |
|---|---|---|---|
| 티켓 생성됨 | ✅ 기본값 | ✅ 기본값 | 클라이언트가 새 지원 티켓을 엽니다 |
| 새 주문 | ✅ 기본값 | ✅ 기본값 | 클라이언트가 새 주문을 합니다 |
| 에이전시 결제 성공 | ✅ 기본값 | ✅ 기본값 | 결제가 성공적으로 처리됩니다 |
| 댓글 추가됨 | ✅ 기본값 | ✅ 기본값 | 작업, 티켓 또는 파일에 새 댓글이 게시됩니다 |
| 파일 피드백 댓글 추가됨 | ✅ 기본값 | ✅ 기본값 | 클라이언트 또는 팀원이 공유 파일에 피드백을 남깁니다 |

<Tip>
  받은 편지함이 너무 많은 알림으로 가득 차면 댓글과 같은 낮은 우선순위 이벤트에 대한 **메일** 알림을 비활성화하고 새 주문 및 결제와 같은 중요한 이벤트에 대해서만 이메일 알림을 유지하세요.
</Tip>

### 기본값으로 재설정

모든 알림 환경설정을 원래 워크스페이스 구성으로 복원하려면 알림 설정 페이지 하단으로 스크롤하여 **기본값으로 재설정**을 클릭합니다.

<Note>
  알림 환경설정은 개인적입니다 — 환경설정을 재설정해도 다른 사용자의 설정에는 영향을 미치지 않습니다. 슈퍼 어드민만 전역 워크스페이스 기본값을 재설정할 수 있습니다.
</Note>

## 프로모션 코드 사용하기

AgencyHandy로부터 프로모션 또는 할인 코드를 받은 경우 계정 설정에서 직접 사용할 수 있습니다.

<Steps>
  <Step title="설정으로 이동">
    왼쪽 사이드바에서 **설정**을 클릭합니다.
  </Step>
  <Step title="프로모션 코드 사용 섹션 열기">
    설정 내에서 **프로모션 코드 사용** 옵션을 찾아 클릭합니다.
  </Step>
  <Step title="코드 입력">
    입력 필드에 프로모션 코드를 입력하거나 붙여넣은 후 **사용**을 클릭합니다.
  </Step>
  <Step title="할인 확인">
    AgencyHandy가 코드를 검증하고 계정에 할인을 적용합니다. 적용된 혜택을 보여주는 확인 메시지가 표시됩니다.
  </Step>
</Steps>
