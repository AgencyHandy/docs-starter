로그인하면 AgencyHandy가 메인 대시보드로 이동시킵니다 — 에이전시가 관리하는 모든 것의 컨트롤 센터입니다. 인터페이스는 깔끔하고 내비게이션 중심으로 설계되어 있습니다: 고정된 왼쪽 사이드바에서 모든 주요 기능에 즉시 접근할 수 있으며, 중앙 패널은 선택한 섹션과 관련된 데이터를 표시하도록 업데이트됩니다. 클라이언트 관계부터 청구서, 태스크 추적까지 에이전시 운영에 필요한 모든 것이 여기에 있습니다.

## 두 가지 대시보드 화면

AgencyHandy는 로그인한 사용자에 따라 두 가지 다른 대시보드 관점을 제공합니다. 필요에 따라 전환할 수 있습니다.

  <Card title="에이전시 대시보드" icon="building">

<Note>
  에이전시 대시보드와 클라이언트 대시보드는 동일한 기본 데이터를 표시하지만 다른 관점과 권한 수준으로 보여집니다. 클라이언트는 자신의 계정과 관련된 데이터만 볼 수 있고, 에이전시 사용자는 모든 클라이언트의 데이터를 볼 수 있습니다.
</Note>

## 왼쪽 사이드바 내비게이션

왼쪽 사이드바는 기본 내비게이션 패널입니다. 각 아이콘과 레이블은 AgencyHandy의 주요 기능 영역에 해당합니다. 각 섹션의 역할은 다음과 같습니다:

### CRM
클라이언트 관계 관리 허브입니다. 모든 클라이언트 연락처 정보를 저장하고 관리하며, 커뮤니케이션 내역을 추적하고, 에이전시가 협력하는 모든 계정의 명확한 기록을 유지합니다. CRM을 클라이언트가 누구이며 각 관계가 어디에 있는지에 대한 단일 정보 출처로 활용하세요.

### 서비스
에이전시가 제공하는 서비스를 정의하고 서비스 카탈로그를 구성하세요. 클라이언트가 주문할 때 여기에 게시한 서비스 중에서 선택합니다. 각 서비스에 대한 가격, 설명, 결과물, 주문 접수 폼을 설정할 수 있습니다.

### 주문
클라이언트 주문이 접수되어 완료될 때까지 모든 주문을 추적하세요. 주문 섹션에서는 주문 상태, 관련 태스크, 클라이언트 커뮤니케이션, 납품 일정을 파악할 수 있어 판매와 납품 사이의 어떤 것도 놓치지 않습니다.

### 제안서
AgencyHandy 내에서 직접 제안서와 계약서를 작성, 발송, 관리하세요. 제안서를 작성하고, 가격과 범위를 첨부하며, 클라이언트에게 검토를 위해 발송하세요. 수락되면 수동 중복 작업 없이 제안서를 활성 주문으로 전환할 수 있습니다.

### 청구서
클라이언트 주문이나 프로젝트에 연결된 전문 청구서를 생성하고, 결제 상태를 추적하며, 클라이언트에게 미납 금액을 명확하게 보여주세요. AgencyHandy는 생성부터 결제까지 청구서 생애주기를 처리하여 청구 관련 주고받기를 줄여줍니다.

### 태스크
태스크 수준에서 모든 작업을 관리하세요. 태스크를 만들고, 팀원에게 할당하며, 마감일과 우선순위를 설정하고, 완료 상태를 모니터링하세요. 태스크는 주문 및 서비스와 직접 연결되어 있어 진행 상황이 항상 맥락 속에서 보입니다.

### 팀
중앙에서 팀원을 관리하세요. 새 멤버를 초대하고, 역할을 할당하며, 각 사람이 워크스페이스 내에서 접근할 수 있는 것을 제어하세요. 에이전시가 성장하면서 팀 구성을 최신 상태로 유지하려면 이 섹션을 활용하세요.

### 티켓
내장된 티켓 시스템을 통해 클라이언트 지원 요청과 내부 이슈 추적을 처리하세요. 클라이언트는 포털에서 티켓을 제출할 수 있으며, 팀은 별도의 헬프데스크 도구 없이 분류, 응답, 해결을 처리할 수 있습니다.

### 타임시트
클라이언트 업무와 내부 프로젝트에 소요된 시간을 추적하세요. 팀원은 특정 태스크나 주문에 대한 시간을 기록할 수 있어 청구, 용량 계획, 프로젝트 수익성 보고에 정확한 데이터를 제공합니다.

### 폼
온보딩, 주문 접수 또는 워크플로의 기타 시점에 클라이언트로부터 구조화된 정보를 수집하기 위한 커스텀 폼을 구성하고 관리하세요. 여기서 만든 폼은 서비스, 제안서에 첨부하거나 클라이언트에게 직접 발송할 수 있습니다.

### 파일
결과물, 계약서, 브랜드 자산, 클라이언트 문서 등 에이전시 업무와 관련된 모든 파일을 하나의 중앙화된 위치에 저장하고 정리하세요. 파일은 특정 클라이언트나 주문에 범위를 지정할 수 있어 모두가 필요한 것이 어디에 있는지 항상 알 수 있습니다.

### 설정
AgencyHandy 워크스페이스의 모든 측면을 구성하세요. 에이전시 프로필, 브랜딩, 알림 설정, 연동, 팀 권한, 청구 등을 관리하세요. 설정은 모든 클라이언트와 팀원에게 적용되는 워크스페이스 수준의 구성을 찾는 곳이기도 합니다.
