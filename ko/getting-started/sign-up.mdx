AgencyHandy 가입은 단 몇 분이면 완료됩니다. 계정을 만들고, 에이전시의 서브도메인이 되는 고유한 워크스페이스 이름을 선택하며, 이메일 주소를 인증하고, 플랫폼이 처음부터 에이전시에 맞게 최적화되도록 간단한 온보딩 과정을 완료하면 됩니다.

<Note>
  시작하기 전에, 가입에 사용할 이메일 받은 편지함에 접근할 수 있는지 확인하세요 — 설정 중 계정 인증에 필요합니다.
</Note>

## 계정 만들기

<Steps>
  <Step title="회원가입 페이지로 이동">
    브라우저를 열고 [app.agencyhandy.com/sign-up](https://app.agencyhandy.com/sign-up)으로 이동하세요. 홈페이지에서 **무료로 시작하기**를 클릭하면 등록 양식으로 이동합니다.
  </Step>

  <Step title="등록 정보 입력">
    회원가입 양식의 모든 필수 항목을 작성하세요:

    | 항목 | 입력 내용 |
    |---|---|
    | **이메일 ID** | 인증에 사용할 수 있는 유효한 이메일 주소 |
    | **이름** | 본인의 이름 |
    | **성** | 본인의 성 |
    | **비밀번호** | 아래 요건을 충족하는 강력한 비밀번호 |
    | **워크스페이스 이름** | 에이전시의 고유 식별자 (아래 규칙 참조) |

    **비밀번호 요건:**
    - 최소 8자 이상
    - 대문자 최소 1개 포함
    - 소문자 최소 1개 포함
    - 숫자 최소 1개 포함
    - 특수 문자 최소 1개 포함 (예: `!`, `@`, `#`)
    - 공백 포함 불가

    **워크스페이스 이름 규칙:**
    - 소문자와 숫자만 사용 가능
    - 문자 사이에 하이픈(`-`) 사용 가능
    - 최소 3자 이상
    - 공백 및 대문자 사용 불가

    <Tip>
      워크스페이스 이름을 신중하게 선택하세요 — 에이전시의 서브도메인의 일부가 되며 브랜드를 나타냅니다. 예를 들어, `bright-agency`라는 워크스페이스 이름은 고유한 AgencyHandy URL의 기반이 됩니다.
    </Tip>
  </Step>

  <Step title="개인정보 처리방침 동의">
    AgencyHandy의 **개인정보 처리방침**에 동의한다는 체크박스를 클릭하세요. 동의하지 않으면 진행할 수 없습니다.
  </Step>

  <Step title="등록 양식 제출">
    **회원가입** 버튼을 클릭하세요. AgencyHandy가 계정을 생성하고 입력한 이메일 주소로 즉시 인증 이메일을 발송합니다.
  </Step>

  <Step title="이메일 주소 인증">
    받은 편지함에서 인증 이메일을 여세요. 고유한 인증 코드가 포함되어 있습니다. AgencyHandy 인증 페이지로 돌아가 지정된 입력란에 코드를 입력하고 **계속**을 클릭하세요.

    <Note>
      몇 분 안에 이메일이 오지 않으면 스팸 또는 정크 폴더를 확인하세요. 인증 코드가 만료된 경우, 인증 페이지에서 **코드 재발송**을 클릭하여 새 코드를 받으세요.
    </Note>

    AgencyHandy가 코드를 수락하면 계정이 확인되고 성공 메시지가 표시됩니다.
  </Step>

  <Step title="에이전시 정보 설정">
    인증 후, 간단한 온보딩 설문이 AgencyHandy를 맞춤 설정하는 데 도움을 줍니다:

    1. **에이전시 유형** — 에이전시를 가장 잘 설명하는 카테고리를 선택하세요 (예: 마케팅, 디자인, 개발, 컨설팅).
    2. **에이전시 사용 목적** — AgencyHandy를 어떻게 사용할 계획인지 간단히 답하세요.
    3. **팀 규모** — 현재 팀의 인원 수를 선택하세요.
    4. **유입 경로** — AgencyHandy를 어떻게 알게 되셨는지 선택하세요.

    각 화면 후 **다음**을 클릭하여 진행하세요.
  </Step>

  <Step title="에이전시 대시보드 도착">
    온보딩 단계를 완료하면 AgencyHandy가 **회사 홈 대시보드**로 이동시킵니다. 이제 워크스페이스가 활성화되어 사용할 준비가 되었습니다.
  </Step>
</Steps>

## Google로 가입하기

별도의 비밀번호를 만들고 싶지 않다면, Google 계정으로 등록할 수 있습니다.

<Steps>
  <Step title="'Google로 계속하기' 클릭">
    회원가입 페이지에서 **Google로 계속하기** 버튼을 클릭하세요.
  </Step>
  <Step title="Google 계정 선택">
    AgencyHandy와 연결할 Google 계정을 선택하고 요청된 권한을 허용하세요.
  </Step>
  <Step title="온보딩 완료">
    AgencyHandy가 Google 프로필 정보를 사용하여 계정을 생성합니다. 대시보드에 도착하기 전에 에이전시 유형 및 팀 규모 온보딩 단계를 안내받게 됩니다.
  </Step>
</Steps>

<Tip>
  Google로 가입한 경우, 매번 로그인할 때 **Google로 계속하기** 버튼을 사용하세요 — Google 이메일이 기본 로그인 ID가 됩니다.
</Tip>

## 중요 사항

<Warning>
  워크스페이스 이름은 지원팀에 문의하지 않고는 나중에 변경할 수 없습니다. 제출하기 전에 에이전시를 정확하게 나타내는지 확인하세요.
</Warning>

- 강력하고 고유한 비밀번호를 사용하고 비밀번호 관리자에 저장하세요.
- 이메일로 발송된 인증 코드는 짧은 시간 후 만료될 수 있습니다. 만료된 경우 **코드 재발송**을 사용하여 새 코드를 요청하세요.
- 온보딩 질문은 특정 기능으로 제한하지 않습니다 — 단순히 초기 대시보드 화면을 개인화하는 데 사용됩니다.
- 대시보드에 도착하면 즉시 클라이언트 추가, 서비스 생성, 첫 번째 제안서 작성을 시작할 수 있습니다.
