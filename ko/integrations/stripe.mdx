AgencyHandy의 Stripe 통합을 통해 플랫폼 내에서 전송하는 인보이스를 통해 클라이언트로부터 직접 카드 결제를 받을 수 있습니다. 연결되면 클라이언트에게 익숙한 Stripe 호스팅 결제창이 표시되며, 결제 상태가 AgencyHandy에 자동으로 동기화됩니다 — 수동 대조가 필요 없습니다.

<Note>
  **슈퍼어드민** 및 **어드민** 역할만 Stripe 통합을 연결하거나 관리할 수 있습니다. 시작하기 전에 적절한 권한이 있는 인증된 Stripe 계정이 있어야 합니다.
</Note>

## Stripe 연결하기

<Steps>
  <Step title="Stripe 통합 설정 열기">
    왼쪽 사이드바에서 **통합 → Stripe 통합**으로 이동합니다.
  </Step>

  <Step title="Stripe에서 인증">
    **Stripe 연결**을 클릭합니다. Stripe 인증 화면으로 리디렉션됩니다. Stripe 계정에 로그인하고 **허용**을 클릭하여 AgencyHandy 접근을 허가합니다. 승인되면 Stripe가 자동으로 AgencyHandy로 다시 리디렉션합니다.
  </Step>

  <Step title="결제 환경설정 구성">
    리디렉션 후 다음 옵션을 설정합니다:

    - **통화** — 워크스페이스의 기본 청구 통화를 선택합니다.
    - **결제 수단** — Stripe 계정에서 지원하는 카드 유형(Visa, Mastercard 등)을 활성화하거나 비활성화합니다.
    - **청구 환경설정** — 선택적으로 세율, 인보이스 접두사 또는 기본 결제 조건 및 만기일을 설정합니다.
  </Step>

  <Step title="저장 및 확인">
    **저장**을 클릭합니다. 통합이 활성화되면 확인 배너가 나타납니다. Stripe가 이제 워크스페이스에 연결되어 클라이언트가 온라인으로 인보이스를 결제할 수 있습니다.
  </Step>
</Steps>

## 통합 테스트하기

라이브 인보이스를 보내기 전에 빠른 엔드투엔드 테스트를 실행하여 연결이 올바르게 작동하는지 확인하세요.

<Steps>
  <Step title="테스트 인보이스 생성">
    AgencyHandy 내에서 임시 인보이스를 생성하고 소액을 지정합니다.
  </Step>
  <Step title="본인에게 전송">
    인보이스를 자신의 이메일 주소나 테스트를 도와줄 팀원에게 전송합니다.
  </Step>
  <Step title="테스트 결제 완료">
    인보이스 링크를 열고 Stripe 계정이 테스트 모드인 경우 Stripe 테스트 카드(예: `4242 4242 4242 4242`)를 사용하거나 라이브 모드인 경우 실제 카드로 결제를 완료합니다.
  </Step>
  <Step title="상태 업데이트 확인">
    AgencyHandy로 돌아가 인보이스 상태가 **결제 완료**로 변경되었는지 확인합니다. 변경되었다면 통합이 올바르게 작동하는 것입니다.
  </Step>
</Steps>

## Stripe 연결 관리하기

언제든지 **통합 → Stripe 통합**으로 돌아가 다음을 수행할 수 있습니다:

- **결제 설정 업데이트** — 통화, 결제 수단 또는 청구 환경설정을 변경합니다.
- **연결 상태 확인** — Stripe가 현재 활성화되어 있는지 확인합니다.
- **연결 해제** — **연결 해제**를 클릭하여 Stripe 계정의 연결을 끊습니다. 재연결할 때까지 결제 수집이 즉시 중단됩니다.
- **재연결** — **Stripe 연결**을 다시 클릭하고 재인증하여 통합을 복원합니다.

<Warning>
  Stripe 연결을 해제하면 모든 미결 인보이스의 결제 수집이 중단됩니다. 재연결하고 재인증할 때까지 클라이언트가 온라인으로 결제할 수 없습니다.
</Warning>

## 추가 참고 사항

<Note>
  AgencyHandy는 결제 자격 증명을 저장하지 않습니다. 모든 결제 데이터는 Stripe의 안전한 인프라에서만 처리됩니다.
</Note>

- **워크스페이스당 하나의 계정** — 각 AgencyHandy 워크스페이스에 단일 Stripe 계정만 연결할 수 있습니다. 계정을 전환하려면 먼저 현재 계정의 연결을 해제한 후 새 계정으로 재연결하세요.
- **라이브 인보이스 상태** — Stripe에 연결된 모든 인보이스는 Stripe에서 직접 수신한 실시간 상태 업데이트를 계속 반영합니다.
- **통화** — AgencyHandy에서 설정한 통화가 Stripe 계정에서 활성화된 통화와 일치하는지 확인하여 결제 오류를 방지하세요.
