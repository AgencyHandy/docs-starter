AgencyHandy의 인보이스 기능은 클라이언트, 주문, 프로젝트를 관리하는 데 사용하는 동일한 플랫폼 내에서 에이전시에 완전한 청구 및 결제 관리 시스템을 제공합니다. 전문적인 인보이스를 생성하고, 이메일로 클라이언트에게 전송하고, 실시간으로 결제 상태를 추적하고, 구독 서비스에 대한 반복 청구를 자동화하고, 필요할 때마다 재무 기록을 내보낼 수 있습니다. 모든 인보이스에는 에이전시 로고와 주소가 자동으로 추가되어 일관된 브랜드 커뮤니케이션을 유지합니다.

## 인보이스를 생성할 수 있는 사람

다음 역할을 가진 사용자만 인보이스를 생성하고 관리할 수 있습니다:

- **슈퍼 어드민**
- **어드민**
- **프로젝트 매니저 (PM)**

클라이언트는 인보이스를 보고 다운로드할 수 있지만 생성하거나 수정할 수는 없습니다.

## 인보이스 대시보드

인보이스 대시보드를 열려면 왼쪽 사이드바에서 **Invoices**를 클릭합니다. 여기서 다음을 수행할 수 있습니다:

- 인보이스 생성, 수정, 삭제
- **클라이언트 이름**, **주문 ID**, 또는 **서비스 이름**으로 인보이스 검색
- 인보이스 목록 필터링 및 정렬
- 한눈에 인보이스 상태 보기
- 접두사 설정을 포함한 인보이스 설정 접근

## 인보이스 상태

모든 인보이스는 정해진 수명 주기를 거칩니다. 이러한 상태를 이해하면 미수금을 효과적으로 관리하는 데 도움이 됩니다.

| 상태 | 설명 |
|--------|-------------|
| **Draft** | 인보이스가 생성되었지만 최종 확정되지 않았습니다. 모든 부분을 수정할 수 있습니다. 준비가 되면 상태를 Open으로 변경하여 확정합니다. 구독과 연결되지 않은 초안 인보이스는 삭제할 수 있습니다. |
| **Open / Unpaid** | 인보이스가 최종 확정되어 결제를 기다리고 있습니다. 에이전시가 이를 클라이언트에게 전송하며, 클라이언트는 보고 결제할 수 있습니다. Open 인보이스를 Paid, Void, 또는 Uncollectable로 변경할 수 있습니다. |
| **Paid** | 인보이스가 결제되었습니다. 인보이스를 Paid로 표시하면 시스템이 결제 날짜를 자동으로 기록합니다. |
| **Void** | 인보이스가 취소되었습니다. 시스템이 무효 날짜를 자동으로 기록합니다. |
| **Uncollectable** | 클라이언트가 결제할 가능성이 낮습니다. 회계 처리에서 대손으로 처리합니다. |

<Note>
  **Send to Client** 버튼을 사용하여 클라이언트에게 인보이스를 전송하면, 상태가 **Draft**에서 **Open**으로 자동 변경됩니다. Open에서 Paid, Void, 또는 Uncollectable로의 상태 변경은 에이전시만 가능합니다.
</Note>

## 직접 인보이스 (주문 ID 없이)

AgencyHandy는 기존 주문에 연결되지 않은 인보이스인 **직접 인보이스**를 지원합니다. 이는 일회성 청구, 비정기 결제, 또는 시스템에 아직 클라이언트로 등록되지 않은 연락처에 청구할 때 유용합니다.

직접 인보이스의 주요 기능:

- 주문 ID를 선택하지 않고 인보이스 전송
- 기존 클라이언트에게 청구하거나 **Independent Email**을 사용하여 모든 이메일 주소로 전송
- 수동으로 항목, 세금, 할인 추가
- 즉시 전송하거나 초안으로 저장

<Tip>
  AgencyHandy에 공식 주문이 없는 작업(긴급 프로젝트, 컨설팅 통화, 범위 외 작업 등)에 대해 클라이언트에게 빠르게 청구해야 할 때 직접 인보이스를 사용하세요.
</Tip>

## 인보이스 내보내기

외부 회계 도구 사용이나 기록 보관을 위해 인보이스 목록을 내보낼 수 있습니다. 내보내기에는 인보이스 ID, 클라이언트 이름, 서비스, 금액, 상태, 날짜 등 모든 표시 열이 포함됩니다. 인보이스 대시보드에서 내보내기 옵션에 접근할 수 있습니다.

## 인보이스 설정

인보이스 대시보드 오른쪽 상단의 **Settings** 버튼에서 글로벌 인보이스 설정을 구성합니다. 여기에 포함되는 항목:

- **인보이스 접두사** — 모든 인보이스 번호 앞에 나타나는 접두사 사용자 지정
- **알림 템플릿** — 미결 인보이스에 대한 자동 결제 알림 설정
