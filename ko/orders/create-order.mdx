AgencyHandy에서 주문을 생성하면 클라이언트에게 서비스를 제공하기 위한 체계적인 워크스페이스가 만들어집니다. 클라이언트, 서비스, 가격, 날짜, 프로젝트 브리프 등 핵심 세부 사항을 미리 입력하면 태스크부터 토론까지 모든 것이 해당 단일 레코드에서 이어집니다. 이 페이지에서는 주문을 처음부터 생성하고, 이후에 업데이트하고, 상태를 변경하고, 더 이상 필요하지 않을 때 삭제하는 방법을 안내합니다.

## 요구 사항

- AgencyHandy에 로그인되어 있어야 합니다.
- 주문을 생성하거나 수정하려면 **슈퍼 어드민**, **어드민**, 또는 **프로젝트 매니저** 역할을 보유해야 합니다.
- 주문 삭제는 **슈퍼 어드민**과 **어드민**만 가능합니다.

## 새 주문 생성

<Steps>
  <Step title="주문 페이지로 이동">
    AgencyHandy 대시보드 왼쪽 사이드바에서 **Orders**를 클릭합니다.
  </Step>

  <Step title="생성 양식 열기">
    오른쪽 상단의 **+ Create New Order**를 클릭합니다. 양식이 나타납니다.
  </Step>

  <Step title="주문 세부 정보 입력">
    양식의 필드를 작성합니다:

    | 필드 | 필수 여부 | 참고 사항 |
    |---|---|---|
    | **Client** | ✅ | 기존 클라이언트 목록에서 선택 |
    | **Service** | ✅ | 게시된 서비스와 미게시 서비스 모두 포함 |
    | **Project Manager** | ❌ | 선택 사항; 납품 담당 PM 지정 |
    | **Quantity** | ✅ | 주문된 수량 |
    | **Price** | ✅ | 주문에 합의된 예산 |
    | **Kick-off Date** | ✅ | 작업 시작 일자 |
    | **Due Date** | ✅ | 납품 마감 일자 |
    | **Order Brief** | ❌ | 팀을 위한 지침, 요구 사항 또는 참고 메모 |
    | **Attachments** | ❌ | 주문과 관련된 지원 파일 |

    <Tip>
      선택한 서비스에 연결된 애드온이 있으면 양식에 **Add-On** 섹션이 나타납니다. 제출 전에 관련 애드온을 포함하세요.
    </Tip>
  </Step>

  <Step title="입력 내용 검토">
    클라이언트, 서비스, 가격, 마감 일자를 특히 꼼꼼히 확인하며 모든 필드의 정확성을 재확인합니다.
  </Step>

  <Step title="주문 제출">
    **Create**를 클릭하여 주문을 완료합니다. 성공적으로 생성되면 확인 메시지가 표시됩니다.
  </Step>
</Steps>

### 주문 생성 후 진행되는 사항

- 회사의 모든 슈퍼 어드민은 새 주문에 자동으로 접근 권한을 갖게 됩니다.
- 클라이언트는 주문이 생성되었다는 알림을 받습니다.
- 주문은 **Pending** 상태로 **Orders** 목록에 나타납니다.

## 주문 수정 또는 업데이트

세부 사항 수정, 가격 조정, 마감 일자 변경, 또는 클라이언트 요청 반영을 위해 언제든지 주문을 업데이트할 수 있습니다.

<Note>
주문을 수정하려면 **슈퍼 어드민**, **어드민**, 또는 **프로젝트 매니저**여야 합니다.
</Note>

<Steps>
  <Step title="주문 목록 열기">
    왼쪽 사이드바에서 **Orders**로 이동합니다.
  </Step>

  <Step title="수정할 주문 선택">
    업데이트할 주문을 찾아 옆의 체크박스를 선택합니다.
  </Step>

  <Step title="수정 버튼 활성화">
    오른쪽 상단에 **Edit** 버튼이 나타납니다. 클릭하면 주문 세부 정보가 수정 모드로 열립니다.
  </Step>

  <Step title="필요한 필드 수정">
    클라이언트 세부 정보, 서비스, 가격, 날짜, 브리프, 또는 배정된 PM 등 변경이 필요한 필드를 업데이트합니다.
  </Step>

  <Step title="변경 사항 저장">
    **Update**를 클릭하여 저장합니다. 업데이트가 성공적으로 완료되었다는 확인 메시지가 표시됩니다.
  </Step>
</Steps>

## 주문 상태 변경

주문 상태는 납품 파이프라인에서 작업이 어느 단계에 있는지를 나타냅니다. AgencyHandy는 다섯 가지 고정 상태를 정의하며, 추가하거나 이름을 변경할 수 없습니다.

| 상태 | 의미 |
|---|---|
| **Pending** | 생성되었으나 아직 시작되지 않음 |
| **Ongoing** | 현재 작업 중 |
| **In Review** | 검토를 위해 제출됨; 승인 대기 중 |
| **Complete** | 납품 완료 및 승인됨 |
| **Cancelled** | 더 이상 진행되지 않음 |

<Note>
주문 상태는 **슈퍼 어드민**, **어드민**, **프로젝트 매니저**만 변경할 수 있습니다. 직원과 클라이언트는 상태를 변경할 수 없습니다.
</Note>

상태를 업데이트하려면 주문을 열고 주문 상세 페이지 상단의 상태 선택기를 사용합니다. 드롭다운에서 새 상태를 선택하고 확인합니다.

## 주문 삭제

주문을 삭제하면 시스템에서 영구적으로 제거됩니다. 주문이 오류로 생성되었거나 중복된 경우와 같이 진정으로 불필요한 경우에만 사용하세요.

<Warning>
주문 삭제는 되돌릴 수 없습니다. 주문을 삭제하면 복구할 수 없습니다. 신중하게 진행하세요.
</Warning>

<Note>
주문 삭제는 **슈퍼 어드민**과 **어드민**만 가능합니다.
</Note>

<Steps>
  <Step title="주문 페이지로 이동">
    왼쪽 사이드바에서 **Orders**를 클릭합니다.
  </Step>

  <Step title="삭제할 주문 찾기">
    목록에서 제거할 주문을 스크롤하거나 검색합니다.
  </Step>

  <Step title="삭제 시작">
    주문에 연결된 **휴지통 아이콘**을 클릭합니다.
  </Step>

  <Step title="삭제 확인">
    확인 메시지가 표시됩니다. 확인을 클릭하면 주문이 영구적으로 삭제됩니다.
  </Step>
</Steps>

삭제 후에는 나머지 주문을 검토하여 모든 것이 정상인지 확인합니다. 삭제된 항목은 더 이상 어떤 목록이나 보고서에도 나타나지 않습니다.

## 주문에 팀 구성원 추가

슈퍼 어드민, 어드민, PM은 언제든지 주문에 직원을 추가할 수 있습니다. 팀 구성원이 추가되면 인앱 알림을 받습니다. 배정된 직원은 주문과 해당 태스크를 볼 수 있지만, 주문 자체를 관리(세부 정보 수정, 상태 변경, 삭제)하는 것은 불가능합니다.
