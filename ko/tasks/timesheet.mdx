AgencyHandy의 타임시트 기능은 팀이 태스크에 소비하는 모든 시간을 추적할 수 있는 중앙 패널을 제공합니다. 모든 태스크에서 라이브 타이머를 시작하거나, 수동 입력으로 사후에 시간을 기록하고, 날짜 범위, 팀 구성원, 또는 서비스별로 데이터를 분석할 수 있습니다. 생산성을 보고하거나 시간 기준으로 청구해야 할 때 몇 번의 클릭으로 모든 것을 CSV 또는 XLSX로 내보낼 수 있습니다. 팀의 책임감을 유지하고 시간 기반 청구를 정확하게 하는 데 필수적인 도구입니다.

## 타임시트 기능을 사용할 수 있는 사람

| 기능 | 슈퍼 어드민 | 어드민 | PM | 직원 |
|---------|:-----------:|:-----:|:--:|:--------:|
| 본인 시간 로그 보기 | ✅ | ✅ | ✅ | ✅ |
| 다른 사람을 위한 시간 기록 | ✅ | ✅ | ✅ | ❌ |
| 시간 항목 수정 | ✅ | ✅ | ✅ | ❌ |
| 타임시트 데이터 내보내기 | ✅ | ✅ | ✅ | ❌ |

<Note>
  시간을 추적하기 전에 태스크가 존재해야 합니다. 시간을 기록하기 전에 주문 내에 태스크가 생성되어 있는지 확인하세요.
</Note>

## 타임시트 보기

<Steps>
  <Step title="타임시트 열기">
    기본 왼쪽 사이드바에서 **Timesheet**를 클릭합니다. 팀의 모든 시간 로그를 나열하는 표가 표시됩니다.

    각 행에는 날짜, 팀 구성원, 태스크 이름, 서비스, 기록된 시간, 항목에 첨부된 메모가 표시됩니다.
  </Step>
</Steps>

## 시간 항목 추가

시간을 기록하는 두 가지 옵션이 있습니다: 진행 중인 작업을 위한 라이브 타이머 또는 이미 완료한 시간을 위한 수동 입력.

<Tabs>
  <Tab title="타이머 시작">
    <Steps>
      <Step title="태스크 선택">
        타임시트 보기에서 작업할 태스크를 찾거나 선택합니다.
      </Step>
      <Step title="타이머 시작 클릭">
        **Start Timer**를 클릭하여 실시간으로 시간 추적을 시작합니다. 타이머는 중지할 때까지 실행됩니다.
      </Step>
      <Step title="완료 시 중지">
        작업을 마치면 **Stop**을 클릭합니다. AgencyHandy가 선택한 태스크에 경과 시간을 자동으로 기록합니다.
      </Step>
    </Steps>

    <Tip>
      진행 중인 작업에는 타이머를 사용하여 정확한 추적을 보장하세요. 사후에 수동으로 시간을 반올림하거나 추정하는 것은 타이머를 실행하는 것보다 덜 정확합니다.
    </Tip>
  </Tab>
  <Tab title="수동 입력">
    <Steps>
      <Step title="+ Add Entry 클릭">
        타임시트 패널에서 **+ Add Entry**를 클릭합니다. 양식이 나타납니다.
      </Step>
      <Step title="항목 세부 정보 입력">
        - **Task** — 이 시간이 적용되는 태스크 선택
        - **Team Member** — 올바른 사람에게 시간 배정 (어드민과 매니저는 다른 사람을 위해 기록 가능)
        - **Time Spent** — 시간과 분 입력
        - **Notes** — 작업한 내용에 대한 맥락 추가
      </Step>
      <Step title="저장">
        **Save**를 클릭하여 항목을 기록합니다. 즉시 타임시트 표에 나타납니다.
      </Step>
    </Steps>
  </Tab>
</Tabs>

## 날짜 범위별 필터링

날짜 필터를 사용하여 특정 기간에 집중합니다.

<Steps>
  <Step title="날짜 필터 열기">
    타임시트 패널 오른쪽 상단의 날짜 필터 컨트롤을 클릭합니다.
  </Step>
  <Step title="범위 선택">
    사전 설정된 범위에서 선택합니다:
    - **Daily** — 오늘의 로그만
    - **Weekly** — 현재 주
    - **Monthly** — 현재 월

    장기 프로젝트 또는 과거 보고의 경우 시작 날짜와 종료 날짜를 입력하여 **Custom** 날짜 범위를 선택합니다.
  </Step>
  <Step title="적용">
    **Apply**를 클릭하여 선택한 기간의 로그로 표를 업데이트합니다.
  </Step>
</Steps>

<Tip>
  내보내기 전에 필터를 적용하여 집중된 보고서를 생성하세요. 필터링되지 않은 전체 타임시트를 내보내면 활성 에이전시의 경우 매우 큰 파일이 생성될 수 있습니다.
</Tip>

## 항목 그룹화

그룹화는 데이터를 제거하지 않고 타임시트 행이 구성되는 방식을 재배열합니다. 로그를 더 빠르게 읽고 분석하는 데 도움이 됩니다.

<Steps>
  <Step title="Group By 드롭다운 열기">
    타임시트 패널 상단의 **Group By** 드롭다운을 클릭합니다.
  </Step>
  <Step title="그룹화 선택">
    다음 옵션에서 선택합니다:

    | 그룹화 옵션 | 표시 내용 |
    |----------------|---------------|
    | **Task Name** | 각 태스크 아래에 그룹화된 모든 시간 항목 |
    | **Team Member** | 각 사람 아래에 그룹화된 모든 항목 |
    | **Service → Task Name** | 서비스별로 그룹화된 항목, 각 서비스 내에서 태스크별로 세분화 |
  </Step>
</Steps>

<Note>
  그룹화는 시각적 변경만 이루어집니다 — 기본 데이터나 내보내기 대상에 영향을 미치지 않습니다. 내보내기에는 항상 모든 표시 행이 포함됩니다.
</Note>

## 타임시트에서 태스크 세부 정보 보기

<Steps>
  <Step title="태스크 이름 클릭">
    타임시트 표에서 **태스크 이름**을 클릭합니다. AgencyHandy가 해당 태스크의 전체 상세 페이지로 이동합니다.
  </Step>
  <Step title="태스크 맥락 검토">
    태스크 상세 페이지에서 완전한 시간 로그 이력, 첨부된 메모, 담당자, 상태 및 기타 태스크 수준 정보를 확인할 수 있습니다. 이 보기를 사용하여 항목을 조사하거나 어떤 작업이 수행되었는지 확인합니다.
  </Step>
</Steps>

## 타임시트 데이터 내보내기

급여 처리, 클라이언트 청구 보고서, 또는 외부 프로젝트 관리 도구를 위해 타임시트를 내보냅니다.

<Steps>
  <Step title="먼저 필터 적용 (권장)">
    날짜 범위 필터를 사용하여 내보내기를 필요한 기간으로 좁힙니다. 이렇게 하면 파일이 집중적이고 관리하기 쉬워집니다.
  </Step>
  <Step title="내보내기 클릭">
    타임시트 패널 상단의 **Export** 버튼을 클릭합니다.
  </Step>
  <Step title="형식 선택">
    다음 중 하나를 선택합니다:
    - **CSV** — Excel, Google Sheets 또는 데이터 파이프라인과 같은 스프레드시트 도구용
    - **XLSX** — 열 서식이 보존된 Excel 형식의 통합 문서용
  </Step>
  <Step title="파일 다운로드">
    파일이 기기에 자동으로 다운로드됩니다. 각 내보내기에는 모든 표시 열(날짜, 팀 구성원, 태스크 이름, 서비스, 기록된 시간, 메모)이 포함됩니다.
  </Step>
</Steps>

<Warning>
  내보내기가 다운로드를 시작하지 않으면 브라우저의 **팝업 차단** 설정이 파일 다운로드를 방해할 수 있습니다. AgencyHandy에 대한 팝업 차단을 비활성화하고 다시 시도하세요.
</Warning>

## 추가 참고 사항

<Accordion title="타이머 대 수동 입력">
  타이머는 활성적인 실시간 작업에 가장 적합합니다. 수동 입력은 타이머 시작을 잊었거나 소급하여 시간을 기록해야 할 때 가장 적합합니다. 두 방법 모두 타임시트 표에서 동일한 로그 항목을 생성합니다.
</Accordion>

<Accordion title="그룹화 대 필터링">
  필터는 표에 표시되는 행을 **줄입니다**. 그룹화는 이미 표시된 행을 **재구성합니다**. 동시에 둘 다 적용할 수 있습니다 — 날짜 범위로 필터링한 후 팀 구성원별로 그룹화하여 해당 기간의 각 사람의 총계를 볼 수 있습니다.
</Accordion>

<Accordion title="내보내기에는 모든 표시 열이 포함됩니다">
  모든 내보내기는 표시 표의 현재 상태를 캡처합니다: 날짜, 사용자, 태스크, 서비스, 시간, 메모. 그룹화 설정을 적용한 경우 내보내기에도 그룹화 구조가 반영됩니다.
</Accordion>

<Accordion title="과거 데이터를 위한 사용자 지정 범위">
  장기 프로젝트 또는 분기별 검토의 경우 사용자 지정 날짜 범위 옵션을 사용하여 사전 설정된 일별/주별/월별 범위를 넘어서는 과거 타임시트 데이터를 가져옵니다.
</Accordion>
