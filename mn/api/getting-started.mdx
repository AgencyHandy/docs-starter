AgencyHandy API нь гадаад систем эсвэл скриптээс лид үүсгэх, клиент удирдах, захиалга шинэчлэх болон бусад үйлдлийг автоматжуулах боломж олгодог. Хүсэлт бүрийг воркспейсийн тохиргооноос үүсгэсэн API түлхүүрээр баталгаажуулдаг. Энэ гарын авлага нь итгэмжлэлийг хэрхэн олж авах, анхны API дуудлагыг хэрхэн хийх талаар алхам алхмаар тайлбарлана.

<Note>
  AgencyHandy воркспейсийнхээ **SuperAdmin** эсвэл **Admin** байж API түлхүүр үүсгэх боломжтой.
</Note>

## Суурь URL

Бүх API хүсэлтийг дараах хаяг руу явуулна:

```
https://api.agencyhandy.com
```

API баримт бичгийн бүх жишээ хүсэлтүүдэд `{{URL}}`-г энэ суурь URL-ээр солих хэрэгтэй.

## 1-р алхам: API түлхүүр үүсгэх

<Steps>
  <Step title="Воркспейсийн Тохиргоог нээх">
    Воркспейсийн тохиргоо руу дараах замаар орно уу:

    ```
    {{workspaceUrl}}/workspace-config?tab=api-key
    ```

    Энэ нь Воркспейсийн Тохиргоо доторх **API Key** табыг нээнэ.
  </Step>

  <Step title="API түлхүүрийг үүсгэж хуулах">
    **Generate** (эсвэл түлхүүр аль хэдийн байгаа бол **Regenerate**) дарна уу. API түлхүүрийг хуулж, нууцтай адил найдвартай газар хадгалаарай. Та хуудасыг орхисны дараа AgencyHandy дахин харуулахгүй.
  </Step>
</Steps>

<Warning>
  API түлхүүрийг дахин үүсгэх нь өмнөх түлхүүрийг шууд хүчингүй болгодог. Хуучин түлхүүрийг ашиглаж буй одоогийн интеграцуудыг шинэ түлхүүрээр шинэчлэх хүртэл ажиллахаа болино.
</Warning>

## 2-р алхам: Компанийн ID-г олох

Ихэнх API эцсийн цэгүүд нь хүсэлт хаана чиглэж буй воркспейсийг тодорхойлох `companyid` толгой шаарддаг. Компанийн ID-г авахын тулд дараах эцсийн цэгийг ашиглана уу.

### Эцсийн цэг

```
GET {{URL}}/accounts/companies
```

### Толгойнууд

| Толгой | Утга |
|---|---|
| `x-api-key` | Таны API түлхүүр |

### Жишээ хүсэлт

```bash cURL
curl --request GET "https://api.agencyhandy.com/accounts/companies" \
  --header "x-api-key: <YOUR_API_KEY>"
```

### Жишээ хариулт

```json
{
  "message": "Companies associated with API token.",
  "companies": [
    {
      "_id": "6525994184e9ddd79853450e",
      "name": "onethread123",
      "logo": "",
      "extraSmallLogo": "",
      "largeLogo": ""
    }
  ]
}
```

<Steps>
  <Step title="Компанийн ID-г олох">
    Хариултаас `companies` массив доторх `_id` талбарыг олно уу. Энэ бол таны **Компанийн ID**.

    ```javascript
    const companyId = response.companies[0]._id;
    // e.g. "6525994184e9ddd79853450e"
    ```
  </Step>

  <Step title="Компанийн ID-г хадгалах">
    Компанийн ID-г бүх дараагийн API хүсэлтүүдэд ашиглахын тулд хадгалана уу. Үүнийг `companyid` толгойд дамжуулна:

    | Толгой | Утга |
    |---|---|
    | `x-api-key` | Таны API түлхүүр |
    | `companyid` | Таны Компанийн ID |
  </Step>
</Steps>

## Баталгаажуулалтын хураангуй

Баталгаажуулалттай API хүсэлт бүр наад зах нь дараахийг шаарддаг:

```
x-api-key: <YOUR_API_KEY>
companyid: <YOUR_COMPANY_ID>
```

API түлхүүр болон Компанийн ID-тэй болсноор AgencyHandy API-ийн бусад хэсгийг ашиглахад бэлэн болно. Доорх эцсийн цэгийн гарын авлагуудын аль нэгийг үргэлжлүүлнэ үү.
