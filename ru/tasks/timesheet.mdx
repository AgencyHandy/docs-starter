Функция табеля учёта рабочего времени в AgencyHandy предоставляет центральную панель для отслеживания каждого часа, который ваша команда тратит на задачи. Вы можете запустить живой таймер по любой задаче, задним числом внести время вручную, а затем анализировать данные по диапазону дат, участнику команды или услуге. Когда нужно отчитаться о производительности или выставить счёт за время, вы можете экспортировать всё в CSV или XLSX за несколько нажатий. Это незаменимый инструмент для поддержания дисциплины в команде и точного выставления счетов за время.

## Кто может использовать функции табеля

| Функция | Супер-администратор | Администратор | PM | Сотрудник |
|---------|:-----------:|:-----:|:--:|:--------:|
| Просмотр собственных журналов времени | ✅ | ✅ | ✅ | ✅ |
| Запись времени за других | ✅ | ✅ | ✅ | ❌ |
| Редактирование записей времени | ✅ | ✅ | ✅ | ❌ |
| Экспорт данных табеля | ✅ | ✅ | ✅ | ❌ |

<Note>
  Прежде чем вести учёт времени, необходимо, чтобы задачи существовали. Убедитесь, что задачи созданы в заказах, прежде чем пытаться регистрировать время.
</Note>

## Просмотр табеля

<Steps>
  <Step title="Откройте табель">
    Нажмите **Timesheet** на основной левой боковой панели. Вы увидите таблицу со всеми журналами времени вашей команды.

    Каждая строка показывает дату, участника команды, название задачи, услугу, зарегистрированное время и любые заметки к записи.
  </Step>
</Steps>

## Добавление записи времени

Для регистрации времени доступны два варианта: живой таймер для текущей работы или ручная запись для уже выполненной работы.

<Tabs>
  <Tab title="Запуск таймера">
    <Steps>
      <Step title="Выберите задачу">
        В представлении табеля найдите или выберите задачу, над которой вы собираетесь работать.
      </Step>
      <Step title="Нажмите «Запустить таймер»">
        Нажмите **Start Timer**, чтобы начать отслеживание времени в реальном времени. Таймер работает до его остановки.
      </Step>
      <Step title="Остановите по завершении">
        Нажмите **Stop** после завершения работы. AgencyHandy автоматически фиксирует прошедшее время по выбранной задаче.
      </Step>
    </Steps>

    <Tip>
      Используйте таймер для текущей работы, чтобы обеспечить точный учёт. Ручное округление или оценка времени постфактум менее точны, чем непрерывная работа таймера.
    </Tip>
  </Tab>
  <Tab title="Ручной ввод">
    <Steps>
      <Step title="Нажмите «+ Add Entry»">
        На панели табеля нажмите **+ Add Entry**. Появится форма.
      </Step>
      <Step title="Заполните данные записи">
        - **Task** — выберите задачу, к которой относится время
        - **Team Member** — назначьте время правильному человеку (Администраторы и Менеджеры могут вести учёт за других)
        - **Time Spent** — введите часы и минуты
        - **Notes** — добавьте любой контекст о выполненной работе
      </Step>
      <Step title="Сохраните">
        Нажмите **Save** для записи. Запись немедленно появится в таблице табеля.
      </Step>
    </Steps>
  </Tab>
</Tabs>

## Фильтрация по диапазону дат

Используйте фильтры дат для фокусировки на конкретном временном периоде.

<Steps>
  <Step title="Откройте фильтры дат">
    Нажмите элементы управления фильтром дат в правом верхнем углу панели табеля.
  </Step>
  <Step title="Выберите диапазон">
    Выберите из предустановленных диапазонов:
    - **Daily** — только сегодняшние записи
    - **Weekly** — текущая неделя
    - **Monthly** — текущий месяц

    Для более длительных проектов или исторической отчётности выберите **Custom** (произвольный) диапазон дат, указав дату начала и окончания.
  </Step>
  <Step title="Применить">
    Нажмите **Apply**, чтобы обновить таблицу с записями за выбранный период.
  </Step>
</Steps>

<Tip>
  Применяйте фильтры перед экспортом для формирования сфокусированных отчётов. Экспорт полного нефильтрованного табеля может создавать очень большие файлы для активных агентств.
</Tip>

## Группировка записей

Группировка переорганизует строки табеля без удаления каких-либо данных. Она помогает быстрее читать и анализировать журналы.

<Steps>
  <Step title="Откройте выпадающий список «Group By»">
    Нажмите выпадающий список **Group By** в верхней части панели табеля.
  </Step>
  <Step title="Выберите группировку">
    Выберите один из следующих вариантов:

    | Вариант группировки | Что отображает |
    |----------------|---------------|
    | **Task Name** | Все записи времени сгруппированы по каждой задаче |
    | **Team Member** | Все записи сгруппированы по каждому участнику |
    | **Service → Task Name** | Записи сгруппированы по услугам, затем разбиты по задачам в каждой услуге |
  </Step>
</Steps>

<Note>
  Группировка — это только визуальное изменение; оно не влияет на базовые данные и на то, что экспортируется. Экспорт всегда включает все видимые строки.
</Note>

## Просмотр деталей задачи из табеля

<Steps>
  <Step title="Нажмите на название задачи">
    В таблице табеля нажмите на любое **название задачи**. AgencyHandy перенаправит вас на полную страницу деталей этой задачи.
  </Step>
  <Step title="Ознакомьтесь с контекстом задачи">
    На странице деталей задачи вы найдёте полную историю журнала времени, прикреплённые заметки, исполнителей, статус и другую информацию на уровне задачи. Используйте это представление для изучения записей или проверки выполненной работы.
  </Step>
</Steps>

## Экспорт данных табеля

Экспортируйте свой табель для расчёта заработной платы, отчётов о выставлении счетов клиентам или внешних инструментов управления проектами.

<Steps>
  <Step title="Сначала примените фильтры (рекомендуется)">
    Используйте фильтр диапазона дат, чтобы сузить экспорт до нужного периода. Это сохранит файл сфокусированным и управляемым.
  </Step>
  <Step title="Нажмите «Экспорт»">
    Нажмите кнопку **Export** в верхней части панели табеля.
  </Step>
  <Step title="Выберите формат">
    Выберите один из вариантов:
    - **CSV** — для табличных инструментов, таких как Excel, Google Sheets, или конвейеров обработки данных
    - **XLSX** — для книг в формате Excel с сохранением форматирования столбцов
  </Step>
  <Step title="Загрузите файл">
    Файл автоматически загружается на ваше устройство. Каждый экспорт включает все видимые столбцы: дата, участник команды, название задачи, услуга, зарегистрированное время и заметки.
  </Step>
</Steps>

<Warning>
  Если экспорт не запускает скачивание, **блокировщик всплывающих окон** вашего браузера может препятствовать загрузке файла. Отключите блокировку всплывающих окон для AgencyHandy и повторите попытку.
</Warning>

## Дополнительные замечания

<Accordion title="Таймер против ручного ввода">
  Таймер лучше всего подходит для активной работы в режиме реального времени. Ручной ввод лучше, когда вы забыли запустить таймер или вам нужно задним числом записать время. Оба метода дают одинаковые записи журнала в таблице табеля.
</Accordion>

<Accordion title="Группировка против фильтрации">
  Фильтры **сокращают** количество отображаемых строк в таблице. Группировка **переорганизует** уже видимые строки. Вы можете применять оба параметра одновременно — отфильтруйте по диапазону дат, затем сгруппируйте по участнику команды, чтобы увидеть итоги каждого человека за этот период.
</Accordion>

<Accordion title="Экспорт включает все видимые столбцы">
  Каждый экспорт отражает текущее состояние видимой таблицы: дату, пользователя, задачу, услугу, время и заметки. Если вы применили настройку группировки, экспорт также отражает структуру группировки.
</Accordion>

<Accordion title="Произвольные диапазоны для исторических данных">
  Для долгосрочных проектов или квартальных проверок используйте опцию произвольного диапазона дат для получения исторических данных табеля, выходящих за пределы предустановленных ежедневных/еженедельных/ежемесячных окон.
</Accordion>
