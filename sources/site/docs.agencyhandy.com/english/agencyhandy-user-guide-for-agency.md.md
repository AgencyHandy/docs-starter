# AgencyHandy User Guide - For Agency

<figure><img src="/files/pBDoIE74Pt3pa2njP9FH" alt=""><figcaption><p>Manage Clients, Teams &#x26; Invoices In Your Digital Agency</p></figcaption></figure>

Welcome to the AgencyHandy User Guide. This User Manual includes all the necessary information to use the system application AgencyHandy. This manual outlines the system functions and capabilities, contingencies, different modes of operation, and step-by-step approaches for system application access and use in descriptive form. This manual contains and uses actual app-based graphical photos for easy understanding. Select any photos and click on them. You will see the original image with much more details.

This user guide format is an ongoing process and adds more information daily. It may be altered if another format is more suitable for the project.
