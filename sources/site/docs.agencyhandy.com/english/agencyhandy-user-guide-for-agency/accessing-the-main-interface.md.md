# Accessing the Main Interface

Once \*\*logged in\*\*, you'll be taken to the \*\*main interface\*\* of AgencyHandy. This is where you can \*\*create services, orders, and tickets, manage\*\* clients, files, and invoices, \*\*assign tasks\*\* to team members, and track progress. This is called a \*\*Dashboard\*\*.

The \*\*main dashboard\*\* is \*\*intuitive\*\* and \*\*easy to use\*\*, with clear and concise navigation. You can access different features and sections of the software by clicking on the icons in the left column from here.

Right now, AgencyHandy has \*\*two dashboard\*\* views.

\* \*\*Agency Dashboard\*\*: The Agency Dashboard in AgencyHandy is a \*\*centralized platform\*\* for \*\*administrators of an agency\*\* to \*\*view and manage\*\* real-time data and analytics, \*\*monitor\*\* services, orders, ticket and invoice progress, allocate resources, and \*\*manage team\*\* members \*\*effectively\*\*.

<figure><img src="/files/XO87cyc4Jh0R1q76GXwb" alt=""><figcaption></figcaption></figure>

\* \*\*Client Dashboard:\*\* The client dashboard is a centralized platform for clients to view and manage real-time data and analytics and monitor orders, tasks, unpaid invoices and ticket progress.

<figure><img src="/files/73JC4k77cbReW0adQQ5a" alt=""><figcaption></figcaption></figure>

That's it! You can use AgencyHandy to manage your CRM, Proposals, Contracts, Invoices, Client Onboarding, Chat, Tasks, Notes, and Files.

If you have \*\*questions or need help\*\*, \*\*contact\*\* the \*\*AgencyHandy support team\*\* using the Chat button at the bottom right corner.
