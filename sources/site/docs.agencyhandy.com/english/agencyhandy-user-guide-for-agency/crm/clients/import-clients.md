Importing clients saves time and reduces manual data entry by seamlessly transferring client information from external sources into AgencyHandy. It facilitates efficient client management and ensures all relevant data is accessible within the platform.

## 

Pre-condition to Use the Feature:

- Users must have an active account on AgencyHandy and be logged in.
 
- Users should have access to client lists in compatible formats, such as CSV (Comma-Separated Values) files.
 

## 

Steps to Use:

- **Access Client Management:** Log in to your AgencyHandy account.
 
- **Navigate to Clients:** Click on the "Clients" section in your account dashboard.
 
- **Initiate Import:** Look for an option to "Import" at the top right corner and click on it.
 

![](https://docs.agencyhandy.com/~gitbook/image?url=https%3A%2F%2F842362573-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FiedbxyF6rrnGBuHuFME1%252Fuploads%252F3NwqnvgY93XtyQdAGerr%252FScreenshot_6.png%3Falt%3Dmedia%26token%3Df4ef74fd-29b3-436e-9428-c92a1d34ff48&width=768&dpr=3&quality=100&sign=5602fc9a&sv=2)

- **Choose & Upload File:** Select the CSV file containing the client list from your local storage and upload it. You can drag & drop the file here.
 

![](https://docs.agencyhandy.com/~gitbook/image?url=https%3A%2F%2F842362573-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FiedbxyF6rrnGBuHuFME1%252Fuploads%252F8z2dRlGLy79nr7eJCgRe%252FScreenshot_7.png%3Falt%3Dmedia%26token%3D863217c2-4c72-41f1-b664-21aafe711632&width=768&dpr=3&quality=100&sign=d8908a88&sv=2)

- **Map Fields:** If prompted, map the fields in the CSV file to corresponding fields in AgencyHandy (e.g., first name, last name, email, country, state, city, zip code).
 

![](https://docs.agencyhandy.com/~gitbook/image?url=https%3A%2F%2F842362573-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FiedbxyF6rrnGBuHuFME1%252Fuploads%252FvyFQjQpb7NfzLRa9D0UV%252FScreenshot_8.png%3Falt%3Dmedia%26token%3D965f44a9-6222-4be7-a079-52f171a0bfe6&width=768&dpr=3&quality=100&sign=dac9c5de&sv=2)

- **Preview and Confirm:** Peview the imported client list to ensure accuracy and completeness. Click the confirm button.
 

![](https://docs.agencyhandy.com/~gitbook/image?url=https%3A%2F%2F842362573-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FiedbxyF6rrnGBuHuFME1%252Fuploads%252FRfAthrHpXKPDE2OAftQl%252FScreenshot_10.png%3Falt%3Dmedia%26token%3De28367c7-5833-49f4-8ad3-38dae58618ee&width=768&dpr=3&quality=100&sign=3a870509&sv=2)

- **Finalize Import:** Click the "Save" button to finalize the import process.
 

![](https://docs.agencyhandy.com/~gitbook/image?url=https%3A%2F%2F842362573-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FiedbxyF6rrnGBuHuFME1%252Fuploads%252F9qDYB7p0Eo8m4EXSjgbE%252FScreenshot_35.png%3Falt%3Dmedia%26token%3D57ceb70e-83b8-4543-ac0c-3315d8fd04e4&width=768&dpr=3&quality=100&sign=4ae54346&sv=2)

- **Confirmation:** Once imported, you will receive a confirmation message indicating that the clients have been successfully imported into AgencyHandy.
 

### 

Important Notes:

- To import your clients, upload your data in our preferred format (Client name, company name, email, country, state, city, and zip code). Just write the information according to the name of this column.
 
- There is no need to provide a column name at the top. Our platform automatically matches your data columns with our system fields. That means inputting a file with the same column name as our system field will save and show the data correspondently.
 

![](https://docs.agencyhandy.com/~gitbook/image?url=https%3A%2F%2F842362573-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FiedbxyF6rrnGBuHuFME1%252Fuploads%252FOURNI3GldIfYIVgxhRfa%252FScreenshot_9.png%3Falt%3Dmedia%26token%3D8a2d877e-61b0-4f6c-931a-6613b5e9be39&width=768&dpr=3&quality=100&sign=7e5d96e7&sv=2)

- You can manually select a column from our dropdown menu for a seamless import process if you need an extra column.
