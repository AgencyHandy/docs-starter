AgencyHandy provides the capability to set up intake forms, which allow users to gather detailed information about the requirements for each order. Intake forms serve as a structured way to collect essential data from clients, ensuring clarity and completeness in project specifications.

### 

Why It's Needed:

Intake forms are essential for gathering comprehensive client information, ensuring that all project requirements are clearly defined and understood. By using intake forms, agencies can streamline communication, minimize misunderstandings, and deliver better client results.

### 

Pre-Condition to use the feature:

To set up an intake form in AgencyHandy, users must have appropriate permissions to access and modify order settings. Additionally, users should clearly understand the information they need to collect from clients for each order.

### 

Steps to Set Up Intake Form:

In AgencyHandy, users have two options for creating an intake form. One is to create it after creating a service, while the other is from the "Form Sections" feature. This flexibility allows users to design intake forms tailored to their specific needs and preferences.

### 

Steps to Create an Intake Form after Creating a Service:

- **Navigate to Order Settings:** Log in to your AgencyHandy account and navigate to the settings for the specific order for which you want to set up an intake form.
 
- **Access Intake Form Settings:** Within the order settings, locate the section related to the intake form or additional information. This may be labelled as "Intake Form," "Requirements," or similar.
 
- **Define Form Fields:** Determine the specific information you need to collect from clients and create corresponding form fields. Common fields may include project description, deadline, budget, desired features, and any other relevant details.
 
- **Customize Field Types:** Choose appropriate field types for each form field based on the type of information you're collecting. Field types may include text fields, dropdown menus, checkboxes, date pickers, and file upload fields.
 
- **Set Field Requirements:** Specify whether each form field is required or optional. Required fields ensure that clients provide essential information before submitting the intake form.
 
- **Preview and Test Form:** Review the completed intake form to ensure it captures all necessary information effectively. Test the form to verify that it functions correctly and submits data as expected.
 
- **Save Changes:** Once satisfied with the intake form setup, save your changes to apply the form to the order settings.
