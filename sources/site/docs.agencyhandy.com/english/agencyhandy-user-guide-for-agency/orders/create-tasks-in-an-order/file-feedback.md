Agency Handy’s File Feedback system helps you and your clients leave precise, visual comments on images, videos, PDFs, and live websites—directly inside tasks and orders. With version control, threaded discussions, and built-in review tools, it’s designed to streamline feedback and approvals in one place.

### 

What You Can Do

- Drop comments on files, frames, or live websites
 
- Add visual annotations (draw, highlight, text, shapes)
 
- Leave timestamped feedback on video files
 
- Pin feedback to a specific version of a file
 
- Preview and annotate live websites with a browser extension
 
- Sort, search, and filter comments for faster reviews
 
- Keep teams and clients aligned with shared threads
 

---

### 

Why It Matters

- No more lost feedback in email threads
 
- See what’s been addressed with resolved tags
 
- Review changes with full version history
 
- Mark files as “Approved” or “Needs Review”
 
- Keep client and internal feedback separate when needed
 

---

### 

Who Can Use This

To leave or manage feedback, users must:

- Have access to tasks or orders
 
- Be assigned roles like Super Admin, Admin, PM, Assignee or Client
 
- Attach a file or website link to a task

Getting Started

1\. Upload a File or Link

- Open a task or order
 
- Click the ➕ icon in the Attachments section or from the bottom right section
 
- Upload your image, PDF, video, or add a website URL

 Commenting & Annotation
 
 - Click anywhere to leave a comment or right-click to draw a shape
 
 - Comments appear in the sidebar, threaded for discussion
 
 - Choose to share comments with the team, client, or both
 
 - All annotations require a comment—no empty markups

 Comment Actions
 
 From the three-dot menu on a comment:

 - Copy the reply
 
 - Delete the comment

### 

Managing File Versions

- Upload a new version by clicking “Upload New Version”
 
- Older versions and feedback are saved for reference
 
- Use the version dropdown to switch between revisions
 
- When opening an older file, you’ll be notified if a newer version exists
 

> 🗑️ Deleting a file version removes its comments. Deleting a task removes all file versions.

### 

Additional Notes

- Comments are version-specific; editing one doesn’t affect others
 
- Page numbers are visible during PDF annotations
 
- Feedback is only available inside tasks or orders—not service modules or profile images
 
- Feedback must be added via tasks, not the File section
 
- Clients can only comment if client review is enabled
