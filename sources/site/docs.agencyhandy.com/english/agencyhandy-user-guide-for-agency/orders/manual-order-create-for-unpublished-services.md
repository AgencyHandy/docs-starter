The feature in AgencyHandy allows users to create manual orders even for services that have not yet been published. This provides greater flexibility in managing orders for services still in development or temporarily unavailable.

### 

**Pre-Conditions**

- **Active Account:** Users must have an active AgencyHandy account.
 
- **Login Required:** Users must be logged in to create manual orders.
 
- **Permissions:** Users must be Super Admin or Admin to access unpublished services for manual orders.
 

### 

**Steps to Use**

1. **Log in to Your AgencyHandy Account:**
 
 - Access your AgencyHandy account using your credentials.
 
 - Go to the Service section to choose any unpublished service
 
 
 ![](https://docs.agencyhandy.com/~gitbook/image?url=https%3A%2F%2F842362573-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FiedbxyF6rrnGBuHuFME1%252Fuploads%252F4UOzQMdkrvIWBl4jsJY9%252FScreenshot_337.png%3Falt%3Dmedia%26token%3Da99a9ae1-4dfd-4aae-97cb-fc4d2006fa2d&width=768&dpr=3&quality=100&sign=6b5a8d43&sv=2)
 
2. **Navigate to the Order Section:**
 
 - From the left column, click on the **Order** section.
 
 
3. **Create a New Order:**
 
 - Click the **\+ Create Order** button to open the order creation form
 
 
 .
 
 ![](https://docs.agencyhandy.com/~gitbook/image?url=https%3A%2F%2F842362573-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FiedbxyF6rrnGBuHuFME1%252Fuploads%252F1DMT7VSJjk4hwtHkSfBn%252FScreenshot_347.png%3Falt%3Dmedia%26token%3D75292ea1-f3fc-4788-a83f-6bdfe7a8b515&width=768&dpr=3&quality=100&sign=8c151713&sv=2)
 
4. **Select Client Name:**
 
 - Choose the client for whom you are creating the order from the dropdown list.
 
 
5. **Select the Service:**
 
 - In the **Service** section, you will see a list of all available services, including both published and unpublished ones.
 
 - Previously, unpublished services were hidden from this list, but now you can view and select them.
 
 
 ![](https://docs.agencyhandy.com/~gitbook/image?url=https%3A%2F%2F842362573-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FiedbxyF6rrnGBuHuFME1%252Fuploads%252F1cDwyVUTylgTBJIKrGkP%252FScreenshot_338.png%3Falt%3Dmedia%26token%3D5a4fd36b-13d0-4349-aa5b-a638f8e35773&width=768&dpr=3&quality=100&sign=1eb71053&sv=2)
 
6. **Complete Order Details:**
 
 - Add any additional order information such as order brief, pricing, duration, and other relevant details.
 
 
7. **Create the Order:**
 
 - Once all details are filled in, click the **Create** button to finalize the manual order.
 
 

### 

**Important Notes**

- **Visibility of Unpublished Services:** Unpublished services are now visible in the service selection list for manual orders, enabling their use in specific cases.
 
- **Order Flexibility:** This feature allows you to manage services that are still under development or not ready for public release but are available for manual orders.
 
- **Client-specific Orders:** Manual orders for unpublished services can be used for clients who need the service before its official launch.
