The Proposal feature allows agencies to create, customize, and send professional proposals to **clients and leads** directly from Agency Handy. It combines service pricing, add-ons, packages, taxes, and approval workflows into a single, trackable document.

---

### 

What You Can Do

- Create proposals for clients and leads
 
- Edit proposal name, ID, company, address, dates, and assignee
 
- Build proposals using editable content templates
 
- Add multiple services with different pricing types
 
- Include packages, add-ons, items, and tax rates
 
- Create packages, add-ons, items, and tax rates directly from the proposal
 
- Apply currencies per service
 
- Save proposals as draft, preview, or send
 
- Share proposals via public link
 
- Track proposal status and client engagement
 
- Allow comments and password-protect proposals
 
- Automatically create orders and invoices after acceptance
 

---

### 

Why It Matters

- Centralizes sales workflows inside one platform
 
- Reduces manual proposal and invoice creation
 
- Ensures pricing accuracy with structured offerings
 
- Improves visibility across sales, finance, and operations
 
- Speeds up deal closure with clear approval flows

### 

Who Can Use This

- Super Admin
 
- Admin

### 

Accessing Proposals

Navigate to:

**Left Sidebar → Proposals**

From here you can:

- View all proposals
 
- Track proposal status (Draft, Sent, Accepted, Pending)
 
- See creation date, send date, expiry date, and view status
 
- Edit, send, share, or delete proposals from the action menu

### 

Saving, Previewing, and Sending

At any time, you can:

- Save the proposal as Draft
 
- Reset all changes
 
- Preview the proposal
 
- Send the proposal to the client or lead
 

Once sent:

- A public link is generated
 
- The proposal status updates automatically
 
- Engagement (viewed / not viewed) is tracked
 

---

### 

After Sending the Proposal

From the proposal view, you can:

- Edit the proposal
 
- Send again
 
- Download the proposal
 
- Share the public link
 
- View comments (if enabled)
 
- Delete the proposal

### 

Public Proposal View

Clients and leads can:

- View proposal details
 
- Review services, pricing, and taxes
 
- Accept the proposal
 
- Leave comments (if enabled)
 
- Access securely using password protection (if enabled)

### 

Additional Notes

- Proposals support both clients and leads
 
- Multiple emails can be added, with one marked as Primary
 
- Add-ons, items, and taxes are tied to individual services
 
- Orders and invoices can be auto-generated after acceptance
 
- Proposal data remains linked to the client or lead profile
 
- Proposal activity is fully trackable inside Agency Handy
