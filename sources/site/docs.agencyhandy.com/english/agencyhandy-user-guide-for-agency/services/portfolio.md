The Portfolio feature in Agency Handy allows Super Admins and Admins to showcase completed projects or services. Portfolios can include detailed descriptions, project files, categories, and live project links, making them an effective tool for demonstrating expertise and attracting potential clients.

### 

**What:**

This feature enables users to create and display project portfolios linked to specific services, highlighting case studies and successful projects to potential clients.

### 

**Purpose:**

- Showcase completed projects to build credibility and attract new clients.
 
- Link portfolios to services for better context and client understanding.
 
- Streamline the process of demonstrating expertise in specific services.
 

### 

**Pre-conditions:**

- Users must have Super Admin or Admin access to create and manage portfolios.
 
- Relevant project files and information must be available for portfolio creation.
 

---

### 

**Steps to Use:**

### 

**Creating a Portfolio:**

1. **Log in to your Agency Handy account.**
 
2. **Navigate to the Services Tab:**
 
 - Click on the **"Services"** option in the left sidebar.
 
 - Select the **"Portfolio"** section.
 
 
3. **Click on "Create Portfolio":**
 
 - Fill in the required fields:
 
 - **Project Name:** Enter a short and clear name (max 32 characters). It's mandatory.
 
 - **Project Description:** Provide a detailed project description (min 100 characters). It's mandatory.
 
 - **Upload Cover Image:** Upload an image file to serve as the cover for the portfolio. It's mandatory.
 
 - **Upload Files:** Upload relevant project files (images, PDFs, etc.).
 
 - The first uploaded image will automatically be set as the cover image.
 
 - The file size is max 1 GB.
 
 
 - **Project Category:**
 
 - Add or select existing categories to group similar projects.
 
 - Categories can be reused across different portfolios.
 
 
 - **Project Link:** Provide a clickable URL for the live project (optional).
 
 - Click the "Next" button.

### 

**Linking a Portfolio to a Service:**

1. **Navigate to Porfolio Details:**
 
 - Go to an existing Portfolio or create a new one.
 
 
2. **Attach Portfolio to Service:**
 
 - Click the "Add to services"
 
 - During portfolio creation or editing, select from the list of available portfolios.
 
 - Linking a portfolio to a service is optional and can be skipped.
 
 
3. **Save the Service:**
 
 - Click **"Update"** to attach the portfolio to the service.

### 

**Viewing and Managing Portfolios:**

1. **Access Portfolio Details Page:**
 
 - Navigate to the Portfolio section and select a portfolio.
 
 - View the portfolio details, including:
 
 - Name
 
 - Description
 
 - Cover image and files
 
 - Project categories
 
 - "View Live Project" link (if provided).
 
 
 
2. **View Portfolios on the Service Page:**
 
 - Portfolios linked to a service will appear as cards on the service details page.
 
 - Clicking the portfolio card will open a modal displaying all portfolio details.
 
 

---

### 

**Important Notes:**

- The first uploaded file in the portfolio will automatically become the cover image.
 
- Categories can be created once and reused for multiple portfolios.
 
- Linking a portfolio to a service is optional but enhances client understanding.
 
- Portfolios can be edited anytime to update project details or add new files.
 
- Portfolio cards on the service page allow clients to view relevant projects in detail.
