The **Timesheet** feature in Agency Handy helps you track the work logs of your agency team. From this panel, you can start a timer for any task, log time manually, and view logged hours by task, member, or service. You can also filter by date range, group entries, and export the data in CSV or XLSX format. It's an essential tool for monitoring productivity and generating time-based reports.

---

### 

Goals

- View and manage work logs across your agency.
 
- Start a timer or manually log time for tasks.
 
- Filter logs by day, week, month, or custom range.
 
- Group logs by task name, member, or service.
 
- Export timesheet data in CSV or XLSX formats.
 
- Access task details directly from the timesheet.
 

---

### 

Pre-Conditions

- You must be logged into your **Agency Handy** account.
 
- **Permissions Required**: Only users with Admin or Manager roles can log/edit time for others or export data.
 
- Tasks must be created beforehand to allow time tracking.
 
- For exports, browser pop-up blockers should be disabled.

### 

Important Notes

- **Permissions Required**: Only users with appropriate access (e.g., Admins or Managers) can export data or log time for others.
 
- **Export Tip**: Filtering before exporting gives you more focused reports.
 
- **Accurate Logging**: Use the timer for ongoing work to ensure precision.
 
- **Grouping is Visual**: It doesn’t affect data, only helps you read things faster.
 
- **Custom Ranges**: For long projects, use custom date filters to see historical data.
 
- **Timer = real-time tracking**, **Manual = fixed entry**
 
- **Filters** reduce what’s shown, **Grouping** changes how it's organized
 
- **Exports** include all visible columns: date, user, task, service, time, and notes
