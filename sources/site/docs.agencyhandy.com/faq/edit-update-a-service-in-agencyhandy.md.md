# Edit/Update a Service in AgencyHandy

FAQ - Edit/Update a Service in AgencyHandy

#### 1. \*\*Who can edit or update a service in AgencyHandy?\*\*

Only Super Admins and Admins with the necessary permissions can edit or update services.

\*\*\*

#### 2. \*\*Where do I find the option to edit my services?\*\*

Log in to your AgencyHandy account, navigate to the "Services" section from your dashboard, and locate the service you want to edit.

\*\*\*

#### 3. \*\*What details can I modify in an existing service?\*\*

You can update the service title, description, package pricing, availability, images, and additional info.

\*\*\*

#### 4. \*\*Can I change the currency of the service while editing?\*\*

No, the currency type is fixed during the initial setup. If you need a different currency, you have to create a new service.

\*\*\*

#### 5. \*\*Is it possible to update the cover image of the service?\*\*

Yes, you can upload a new image during the editing process to enhance the service listing.

\*\*\*

#### 6. \*\*How do I initiate the editing process?\*\*

Click the 3-dot button next to the service listing, then click the "Edit" option from the menu.

\*\*\*

#### 7. \*\*Will clients be notified if I update the service details?\*\*

No, clients are not automatically notified. It is recommended to inform them manually if changes are significant.

\*\*\*

#### 8. \*\*Can I add more packages to a service during editing?\*\*

Yes, you can add more packages to the service if it has not yet reached the maximum limit of 5 packages.

\*\*\*

#### 9. \*\*How do I save changes after editing the service?\*\*

Once you finish editing, click the "Update" button to save all changes.

\*\*\*

#### 10. \*\*Will the updates reflect immediately after saving?\*\*

Yes, all changes will reflect immediately once saved.

\*\*\*

#### 11. \*\*Can I delete a package while editing the service?\*\*

Yes, you can remove a package during the editing process without affecting the main service.

\*\*\*

#### 12. \*\*Is there an option to preview changes before saving?\*\*

Currently, there is no dedicated preview option, but you can view the changes directly on the dashboard once updated.

\*\*\*

#### 13. \*\*Can I modify the service description during editing?\*\*

Yes, you can update the service description to reflect new offerings or improvements.

\*\*\*

#### 14. \*\*What happens if I close the edit window without saving?\*\*

If you close the window or navigate away, any unsaved changes will be lost.

\*\*\*

#### 15. \*\*Can I update the duration of the service while editing?\*\*

Yes, you can adjust the duration settings during the editing process.

\*\*\*

#### 16. \*\*Will changes in service pricing affect existing clients?\*\*

No, existing clients under a subscription will not be affected. Only new purchases will reflect the updated pricing.

\*\*\*

#### 17. \*\*Can I edit the tasks associated with the service?\*\*

Yes, you can add, remove, or modify default tasks associated with the service during editing.

\*\*\*

#### 18. \*\*How do I confirm that my changes are saved successfully?\*\*

After clicking "Update," you will see a confirmation message indicating that the service listing has been successfully updated.

\*\*\*

#### 19. \*\*Is there a log of changes made to a service?\*\*

Currently, AgencyHandy does not provide a version history or change log for services.

\*\*\*

#### 20. \*\*Can I undo changes after clicking 'Update'?\*\*

No, once you click "Update," the changes are saved permanently. You would need to manually revert them if needed.

\*\*\*

#### 21. \*\*How do I update media files like images in the service?\*\*

During editing, you can replace existing media files by uploading new ones directly.

\*\*\*

#### 22. \*\*Can I set a different trial period when editing the service?\*\*

Yes, you can adjust the trial period of the service during the editing phase.

\*\*\*

#### 23. \*\*What happens if I exceed the image upload size limit during editing?\*\*

The platform will not accept the image. Ensure it is under 5 MB and in the correct format (PNG, JPG, JPEG).

\*\*\*

#### 24. \*\*How do I unpublish a service during the editing process?\*\*

You can toggle the "Publish" option off during editing if you want to temporarily hide the service.

\*\*\*

#### 25. \*\*Can I edit the service name after publishing?\*\*

Yes, the service name can be edited at any time, and it will reflect immediately.

\*\*\*

#### 26. \*\*Do I need to republish the service after editing?\*\*

No, the service remains live. All changes are automatically applied without republishing.

\*\*\*

#### 27. \*\*Is it possible to archive a service instead of deleting it?\*\*

Currently, AgencyHandy only supports publishing, unpublishing, or deleting services. Archiving is not available.

\*\*\*

#### 28. \*\*Can I update the FAQ section during service editing?\*\*

Yes, you can add, remove, or modify FAQs associated with the service.

\*\*\*

#### 29. \*\*Will clients see the edits in real-time?\*\*

Yes, once saved, the edits are visible to clients in real-time.

\*\*\*

#### 30. \*\*Can I edit the tags or categories for the service?\*\*

Yes, you can update the tags or categories to improve search visibility during the editing process.

\*\*\*

#### 31. \*\*Can I remove the intake form during service editing?\*\*

Yes, you can unlink or remove the intake form during the editing phase.

\*\*\*

#### 32. \*\*Will updating a service affect its linked projects?\*\*

No, editing a service will not impact ongoing projects linked to it.

\*\*\*

#### 33. \*\*Can I temporarily disable a service during editing?\*\*

Yes, you can unpublish it while making edits and republish it when ready.

\*\*\*

#### 34. \*\*What if I want to revert back to the old details after updating?\*\*

You would need to manually re-enter the previous details, as there is no revert option.

\*\*\*

#### 35. \*\*Are service updates synchronized with client portals automatically?\*\*

Yes, any update to the service is reflected in the client portal automatically.

\*\*\*
