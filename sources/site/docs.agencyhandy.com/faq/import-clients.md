**AgencyHandy – Import Clients Feature FAQs**

---

**1\. What is the purpose of the Import Clients feature in AgencyHandy?**

The Import Clients feature allows users to seamlessly transfer client information from external sources into AgencyHandy, saving time and reducing manual data entry.

---

**2\. Who can use the Import Clients feature?**

Only users with an active AgencyHandy account and appropriate permissions can access and use the Import Clients feature.

---

**3\. What file formats are supported for importing clients?**

AgencyHandy supports importing client data using CSV (Comma-Separated Values) files.

---

**4\. How do I access the Import Clients feature?**

Log in to your AgencyHandy account, navigate to the "Clients" section, and click on the "Import" option located at the top right corner.

---

**5\. What information should be included in the CSV file?**

The CSV file should contain client details such as client name, company name, email, country, state, city, and zip code.

---

**6\. Is it necessary to include column headers in the CSV file?**

No, there is no need to provide column names at the top. AgencyHandy automatically matches your data columns with its system fields.

---

**7\. Can I manually map fields during the import process?**

Yes, if prompted, you can manually map the fields in your CSV file to the corresponding fields in AgencyHandy.

---

**8\. What happens after I upload the CSV file?**

After uploading, you'll have the opportunity to preview the imported client list to ensure accuracy before finalizing the import. [docs.agencyhandy.com](https://docs.agencyhandy.com/en-us/agencyhandy-user-guide-for-agency/crm/clients/import-clients?utm_source=chatgpt.com)

---

**9\. How do I finalize the import process?**

---

**10\. Will I receive a confirmation after importing clients?**

Yes, upon successful import, a confirmation message will indicate that the clients have been successfully added to AgencyHandy. [docs.agencyhandy.com](https://docs.agencyhandy.com/en-us/agencyhandy-user-guide-for-agency/crm/clients/import-clients?utm_source=chatgpt.com)

---

**11\. Can I import clients without an email address?**

No, an email address is required for each client to ensure proper communication and account setup.

---

**12\. What should I do if the system doesn't recognize a column in my CSV file?**

You can manually select the appropriate column from the dropdown menu during the field mapping step to ensure accurate data import. [docs.agencyhandy.com](https://docs.agencyhandy.com/en-us/agencyhandy-user-guide-for-agency/crm/clients/import-clients?utm_source=chatgpt.com)

---

**13\. Is there a limit to the number of clients I can import at once?**

There is no specified limit; however, it's advisable to ensure your CSV file is well-structured and free of errors for a smooth import process.

---

**14\. Can I import additional client details beyond the basic fields?**

Yes, you can include extra columns in your CSV file, and during the import process, you can map these to corresponding fields in AgencyHandy. [docs.agencyhandy.com](https://docs.agencyhandy.com/en-us/agencyhandy-user-guide-for-agency/crm/clients/import-clients?utm_source=chatgpt.com)

---

**15\. What happens if there are duplicate entries in my CSV file?**

AgencyHandy will handle duplicates based on the email address. It's recommended to ensure each client has a unique email to prevent duplication.

---

**16\. Can I undo an import if I realize there was an error?**

Currently, there's no direct undo feature. You would need to manually delete the imported clients or correct the data as necessary.

---

**17\. Will imported clients receive any notification?**

Yes, clients will receive an email containing a secure login URL, allowing them to access their profile on AgencyHandy.

---

**18\. Can I import clients into specific categories or groups?**

While the import process focuses on client details, you can later categorize or group clients within AgencyHandy as needed.

---

**19\. Is it possible to schedule regular imports?**

AgencyHandy currently supports manual imports. For automated or scheduled imports, consider integrating with other tools or using APIs if available.

---

**20\. What should I do if I encounter errors during the import?**

Review the error messages provided, ensure your CSV file is correctly formatted, and that all required fields are filled. If issues persist, contact AgencyHandy support.

---

**21\. Can I import clients from other CRM platforms?**

Yes, as long as you can export your client data into a CSV format, you can import it into AgencyHandy.

---

**22\. Does AgencyHandy provide a sample CSV template for imports?**

While not explicitly mentioned, it's advisable to refer to the preferred format: client name, company name, email, country, state, city, and zip code. [docs.agencyhandy.com](https://docs.agencyhandy.com/en-us/agencyhandy-user-guide-for-agency/crm/clients/import-clients?utm_source=chatgpt.com)

---

**23\. Are there any data validation checks during import?**

Yes, during the preview step, you can verify the accuracy and completeness of the data before finalizing the import.

---

**24\. Can I import clients with international addresses?**

Yes, ensure that the country, state, city, and zip code fields are correctly filled to accommodate international addresses.

---

**25\. Where can I find more detailed guidance on importing clients?**

For a comprehensive guide, refer to the official AgencyHandy documentation on importing clients: [docs.agencyhandy.com](https://docs.agencyhandy.com/en-us/agencyhandy-user-guide-for-agency/crm/clients/import-clients?utm_source=chatgpt.com).
