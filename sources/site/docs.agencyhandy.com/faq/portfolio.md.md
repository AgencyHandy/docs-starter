# Portfolio

#### \*\*Agency Handy Portfolio Feature – Frequently Asked Questions\*\*

\*\*\*

\*\*1. What is the Portfolio feature in Agency Handy?\*\*

The Portfolio feature allows Super Admins and Admins to showcase completed projects or services by creating detailed portfolios, including descriptions, files, categories, and live project links.

\*\*\*

\*\*2. Who can create and manage portfolios in Agency Handy?\*\*

Only users with Super Admin or Admin access can create and manage portfolios within Agency Handy.

\*\*\*

\*\*3. What kind of content can I include in a portfolio?\*\*

A portfolio can include a project name, description, cover image, relevant files (like images or PDFs), project categories, and a live project link.

\*\*\*

\*\*4. Can I link multiple portfolios to a single service?\*\*

Yes, you can link multiple portfolios to a single service to showcase various related projects.

\*\*\*

\*\*5. Is it mandatory to link a portfolio to a service?\*\*

No, linking a portfolio to a service is optional, but it is recommended for better client context and understanding.

\*\*\*

\*\*6. How do I create a new portfolio?\*\*

Navigate to the \*\*Services\*\* tab, click on \*\*Portfolio\*\*, and select \*\*Create Portfolio\*\*. Fill in the required details and upload relevant files, then save the portfolio.

\*\*\*

\*\*7. What is the character limit for the project name and description?\*\*

The project name must be a maximum of 32 characters, while the project description must be at least 100 characters.

\*\*\*

\*\*8. Can I upload multiple files to a single portfolio?\*\*

Yes, you can upload multiple files, including images and PDFs, with a maximum file size of 1 GB each.

\*\*\*

\*\*9. Which file will be set as the cover image?\*\*

The first uploaded image file will automatically be set as the cover image for the portfolio.

\*\*\*

\*\*10. Can I categorize portfolios for better organization?\*\*

Yes, you can add or select existing categories while creating a portfolio. These categories help group similar projects.

\*\*\*

\*\*11. Can I reuse the same category across different portfolios?\*\*

Yes, categories created once can be reused across multiple portfolios for consistency.

\*\*\*

\*\*12. How do I link a portfolio to a specific service?\*\*

While creating or editing a portfolio, use the \*\*Add to Services\*\* option to select the relevant services to link.

\*\*\*

\*\*13. Can I add a live project link to my portfolio?\*\*

Yes, you can add a clickable URL to the portfolio, allowing clients to view the live project.

\*\*\*

\*\*14. Where can I view all created portfolios?\*\*

You can view all portfolios under the \*\*Portfolio\*\* section within the \*\*Services\*\* tab.

\*\*\*

\*\*15. How do clients view linked portfolios?\*\*

Portfolios linked to services will appear as cards on the service details page. Clicking the card opens a modal displaying full portfolio details.

\*\*\*

\*\*16. Can I edit or update an existing portfolio?\*\*

Yes, you can edit any existing portfolio by accessing it through the \*\*Portfolio\*\* section and making the necessary changes.

\*\*\*

\*\*17. What happens if I upload multiple images to a portfolio?\*\*

The first image you upload will automatically be the cover image, but you can add additional images to the portfolio.

\*\*\*

\*\*18. Can I update a portfolio after it has been created?\*\*

Yes, you can update the portfolio anytime, including adding new files, changing the cover image, or modifying project details.

\*\*\*

\*\*19. What if I want to unlink a portfolio from a service?\*\*

Edit the portfolio and uncheck the linked service from the list. Save the changes to update the linkage.

\*\*\*

\*\*20. How do I delete a portfolio?\*\*

Navigate to the \*\*Portfolio\*\* section, select the portfolio you want to remove, and choose the \*\*Delete\*\* option.

\*\*\*

\*\*21. Can I view the portfolio details without linking it to a service?\*\*

Yes, portfolios exist independently and can be accessed from the \*\*Portfolio\*\* section even if not linked to any service.

\*\*\*

\*\*22. What should I do if my uploaded file size exceeds the limit?\*\*

Reduce the file size to within the 1 GB limit or compress the file before uploading.

\*\*\*

\*\*23. What type of file formats are supported in the portfolio?\*\*

You can upload various file formats, including images (JPG, PNG) and documents (PDF).

\*\*\*

\*\*24. Is it possible to change the cover image after the portfolio is created?\*\*

Yes, you can update the cover image by editing the portfolio and uploading a new image as the first file.

\*\*\*

\*\*25. How can I check if my portfolio is linked correctly to a service?\*\*

Visit the service details page to see if the portfolio appears as a linked card. Click the card to view the full portfolio.

\*\*\*

\*\*26. What if I want to showcase multiple related projects in one portfolio?\*\*

Group related projects within one portfolio by including detailed descriptions and multiple files.

\*\*\*

\*\*27. Can clients see the portfolios directly from their dashboard?\*\*

No, clients access portfolios linked to services through the service details page.

\*\*\*

\*\*28. How can I ensure that portfolios are visually appealing?\*\*

Use high-quality images as the cover and include well-structured project descriptions to make the portfolio attractive.

\*\*\*

\*\*29. What should I include in the project description?\*\*

Include the project objective, key achievements, client feedback (if available), and a brief overview of the service provided.

\*\*\*

\*\*30. Can I preview how the portfolio will appear before publishing?\*\*

Yes, you can view the portfolio in the \*\*Portfolio\*\* section to see how it looks to clients.
