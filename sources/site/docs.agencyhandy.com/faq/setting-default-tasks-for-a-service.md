FAQ - Setting Default Tasks for a Service in Agency Handy

**1\. What are the default tasks in Agency Handy?**

Default tasks are predefined tasks automatically created when a client purchases a service. They help maintain consistency and streamline service management.

**2\. Why should I use default tasks for my services?**

Using default tasks ensures that specific actions are automatically set up whenever the service is purchased, saving time and reducing manual entry.

**3\. Can I set default tasks for every service I create?**

Yes, default tasks can be configured for each service during the creation process in Agency Handy.

**4\. How do I add default tasks while creating a service?**

Go to the 'Services' section, click on '+ Create Service,' and navigate to the 'Additional Info' section. There you will find an option to add default tasks.

**5\. Can I edit the default tasks after setting them up?**

Yes, you can update or modify default tasks at any time through the service configuration settings.

**6\. Is it possible to add multiple default tasks for one service?**

Yes, you can add multiple tasks as needed when setting up the service.

**7\. Are default tasks visible to clients?**

By default, clients can view the tasks from their portal when exploring orders.

**8\. Can I assign different default tasks to different service packages?**

Yes, each package within a service can have its own set of default tasks, allowing for tailored workflows.

**9\. Can default tasks be deleted after they are set?**

Yes, you can remove or modify default tasks by editing the service settings.

**10\. Can I add subtasks within a default task?**

Currently, only main tasks can be set as default. Subtasks must be added manually after the primary task is created.

**11\. Are there any limits on the number of default tasks per service?**

There is no set limit on the number of default tasks you can add.

**12\. Do I need to reconfigure default tasks if I duplicate a service?**

No, duplicated services will automatically include the default tasks from the original service.

**13\. What happens if I remove a default task from a live service?**

The change will only affect future purchases of the service. Existing clients will not be impacted.

**14\. How do I know if my default tasks are correctly set up?**

You can preview the default tasks during the service setup and make adjustments as needed.

**15\. Can default tasks be updated in bulk for multiple services?**

Currently, bulk updates are not supported. You need to configure each service individually.

**16\. Will the client receive a notification when default tasks are created?**

This depends on the notification settings within your workspace. By default, notifications can be configured for task creation.

**17\. Can I link default tasks to specific team members?**

Yes, you can assign default tasks to team members during the service configuration process.

**18\. Are there templates available for common default tasks?**

You can create task templates and reuse them across multiple services.

**19\. Can I track the progress of default tasks separately from custom tasks?**

Yes, default tasks are integrated into the task management system, allowing for tracking and reporting.

**20\. Do default tasks automatically update when the service is edited?**

If you update the service settings, you will need to manually update the default tasks to reflect any changes.

**21\. How do I remove default tasks from an existing service?**

Go to the service settings, locate the default tasks section, and delete the tasks you no longer need.

**22\. Can default tasks be time-bound or recurring?**

Yes, you can set deadlines and recurrence patterns when configuring each task.

**23\. Are there any limitations on the types of tasks I can set as default?**

You can set any standard task as a default task. However, complex task structures may require manual setup.

**24\. What should I do if default tasks are not being created properly?**

Check the service configuration to ensure tasks are correctly saved. If issues persist, contact Agency Handy support.
