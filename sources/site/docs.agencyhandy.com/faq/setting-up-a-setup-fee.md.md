# Setting up a Setup Fee

#### FAQ: Setting up a Setup Fee in Agency Handy

1. \*\*What is the Setup Fee feature in Agency Handy?\*\*
 \* The Setup Fee feature allows agencies to charge an initial one-time or recurring fee for subscription-based services during the initial purchase process.
2. \*\*Is the Setup Fee applied only for subscription-based services?\*\*
 \* Yes, the Setup Fee can only be applied to subscription-based services in Agency Handy.
3. \*\*Can I customize the amount of the Setup Fee?\*\*
 \* Absolutely. You can specify the exact amount of the Setup Fee when configuring the service.
4. \*\*Where do I configure the Setup Fee?\*\*
 \* You can configure the Setup Fee under the 'Package Details' section of the service setup form.
5. \*\*Can the Setup Fee be applied on a recurring basis?\*\*
 \* Yes, the Setup Fee can be set to recur for multiple billing cycles if needed.
6. \*\*How many cycles can the Setup Fee be applied for?\*\*
 \* The duration of the Setup Fee cannot exceed the overall subscription cycle duration.
7. \*\*Does the Setup Fee show up in the catalog's pricing range?\*\*
 \* No, the Setup Fee is not displayed in the catalog's visible pricing range. It is shown separately during checkout.
8. \*\*Will clients see the Setup Fee during checkout?\*\*
 \* Yes, the Setup Fee will be clearly displayed during the checkout process as a separate line item.
9. \*\*Can the Setup Fee be added to manual orders?\*\*
 \* Yes, but it must be added manually during the manual order creation process.
10. \*\*Does the Setup Fee appear separately on invoices?\*\*

\* Yes, the Setup Fee appears as a separate line item from the subscription fee on the invoice.

11. \*\*Can the Setup Fee be edited in a proposal form?\*\*

\* No, once configured, the Setup Fee cannot be edited in the proposal form. It is automatically applied based on the predefined settings.

12. \*\*How do I activate the Setup Fee for a service?\*\*

\* During service configuration, toggle the 'Enable Setup Fee' option to activate it.

13. \*\*What happens if I forget to manually add the Setup Fee in a manual order?\*\*

\* The client will not be charged for the Setup Fee, so it's crucial to remember to add it manually.

14. \*\*Is the Setup Fee refundable if the client cancels the service?\*\*

\* This depends on your agency's refund policy. Agency Handy does not automatically process refunds for Setup Fees.

15. \*\*Can the Setup Fee be hidden from clients?\*\*

\* No, for transparency, the Setup Fee is always displayed during checkout.

16. \*\*Can I use different currencies for the Setup Fee?\*\*

\* Yes, you can select the currency when configuring the Setup Fee amount.

17. \*\*Does the Setup Fee affect the total subscription cost?\*\*

\* Yes, it is added to the initial payment, increasing the total cost displayed at checkout.

18. \*\*Will the Setup Fee apply to renewals?\*\*

\* No, the Setup Fee is only applied during the initial setup unless configured to recur for specific billing cycles.

19. \*\*Is the Setup Fee calculated before or after tax?\*\*

\* The Setup Fee is calculated before tax, and the tax is applied to the total amount, including the Setup Fee.

20. \*\*Can I disable the Setup Fee after enabling it?\*\*

\* Yes, you can disable it by toggling off the 'Enable Setup Fee' option during service configuration.

21. \*\*Can I view reports for collected Setup Fees?\*\*

\* Yes, collected Setup Fees are reflected in your service reports for financial tracking.

22. \*\*What happens if I update the Setup Fee after it has been charged?\*\*

\* Updates only affect future subscriptions; past invoices and payments remain unchanged.
