### 

Appearance Settings in Agency Handy – Frequently Asked Questions

#### 

1\. **What are the Appearance settings in Agency Handy?**

The Appearance settings allow agencies to customize visual elements of their platform, including primary color, sidebar background color, sidebar text color, and sidebar hover background color. This customization ensures a branded and visually cohesive user interface across the platform. [en-us | User Guide](https://docs.agencyhandy.com/workspace-config/set-up-appearance?utm_source=chatgpt.com)

#### 

2\. **Why should I customize the Appearance settings?**

Customizing the Appearance settings helps in:

- **Brand Consistency**: Ensures the platform reflects your agency’s branding with consistent colors and themes.
 
- **Enhanced User Experience**: Creates a visually appealing interface for better navigation and interaction.
 
- **Visual Hierarchy**: Guides user focus on important elements, improving usability with strategic color choices.
 

#### 

3\. **Who can modify the Appearance settings?**

Only users with administrative access or the appropriate permissions (Super Admins and Admins) can access and modify the Appearance settings.

#### 

4\. **How do I access the Appearance settings?**

To access the Appearance settings:

1. Log in to your Agency Handy account.
 
2. Navigate to the **Workspace Settings** section.
 
3. Locate and click on the **Appearance** or **Visual Settings** option. [en-us | User Guide](https://docs.agencyhandy.com/workspace-config/set-up-appearance?utm_source=chatgpt.com)
 

#### 

5\. **What customization options are available in the Appearance settings?**

The available customization options include:

- **Primary Color**: Affects buttons and links for consistency in the visual theme.
 
- **Sidebar Background Color**: Defines the background color of the left-hand navigation sidebar.
 
- **Sidebar Text Color**: Controls the color of text displayed in the sidebar.
 
- **Sidebar Hover Background Color**: Changes the background color when hovering over sidebar elements.
 

#### 

6\. **How do I set the Primary Color?**

In the Appearance settings:

1. Select the **Primary Color** that represents your brand.
 
2. This color will be applied to buttons and links throughout the platform. [en-us | User Guide](https://docs.agencyhandy.com/2.-getting-started/2.5-workspace-config?utm_source=chatgpt.com)[en-us | User Guide](https://docs.agencyhandy.com/workspace-config/set-up-appearance?utm_source=chatgpt.com)
 

#### 

7\. **How can I configure the Sidebar Background Color?**

In the Appearance settings:

1. Choose the **Sidebar Background Color** for the workspace.
 
2. This color will set the background of the sidebar for a cohesive theme. [en-us | User Guide](https://docs.agencyhandy.com/workspace-config/set-up-appearance?utm_source=chatgpt.com)
 

#### 

8\. **How do I adjust the Sidebar Text Color?**

In the Appearance settings:

1. Select the **Sidebar Text Color** to ensure readability.
 
2. Choose a color that contrasts nicely with the sidebar background. [en-us | User Guide](https://docs.agencyhandy.com/workspace-config/set-up-appearance?utm_source=chatgpt.com)
 

#### 

9\. **How can I define the Sidebar Hover Background Color?**

In the Appearance settings:

1. Specify the **Sidebar Hover Background Color** for sidebar elements.
 
2. This color will appear when users hover over sidebar items, enhancing interactivity. [en-us | User Guide](https://docs.agencyhandy.com/workspace-config/set-up-appearance?utm_source=chatgpt.com)
 

#### 

10\. **Can I preview the changes before applying them?**

Yes, after configuring the desired settings, you can preview them to ensure they align with your branding guidelines. Once satisfied, click the **Save Changes** button to apply the new appearance settings to your workspace. [en-us | User Guide](https://docs.agencyhandy.com/workspace-config/set-up-appearance?utm_source=chatgpt.com)

#### 

11\. **Will the appearance changes affect all users in the workspace?**

Yes, changes in appearance will be applied globally across the workspace, impacting all users.

#### 

12\. **Is there an option to reset the Appearance settings to default?**

Yes, clicking the **Reset Default Settings** button will revert all appearance changes back to the system’s default theme. Please note that this action cannot be undone, so ensure you review changes before applying. [en-us | User Guide](https://docs.agencyhandy.com/workspace-config/set-up-appearance?utm_source=chatgpt.com)

#### 

13\. **What are the best practices for customizing the Appearance settings?**

- Use high-contrast colors for sidebar text to ensure readability.
 
- Choose a primary color that aligns with your brand identity.
 
- Adjust hover colors to improve user interaction feedback.
 
- Preview your selections before finalizing changes to maintain a professional look. [en-us | User Guide](https://docs.agencyhandy.com/workspace-config/set-up-appearance?utm_source=chatgpt.com)
 

#### 

14\. **Are there any general recommendations for managing Appearance settings?**

- Ensure colours are visually accessible for all users, avoiding extreme contrasts.
 
- Keep a record of your brand’s colour palette in case settings need to be reapplied.
 
- If unsure about colour choices, test different combinations before finalising.
