เมื่อคุณล็อกอิน AgencyHandy จะพาคุณไปยังแดชบอร์ดหลัก — ศูนย์ควบคุมสำหรับทุกสิ่งที่เอเจนซี่ของคุณจัดการ อินเทอร์เฟซสะอาดตาและขับเคลื่อนด้วยการนำทาง: แถบด้านข้างซ้ายที่แสดงอยู่ตลอดเวลาให้คุณเข้าถึงฟีเจอร์หลักทุกอย่างได้ทันที ขณะที่แผงกลางจะอัปเดตเพื่อแสดงข้อมูลที่เกี่ยวข้องกับส่วนที่คุณเลือก ทุกสิ่งที่คุณต้องการเพื่อบริหารเอเจนซี่ — ตั้งแต่ความสัมพันธ์กับลูกค้าไปจนถึงใบแจ้งหนี้และการติดตามงาน — อยู่ที่นี่

## การนำทางแถบด้านข้างซ้าย

แถบด้านข้างซ้ายคือแผงนำทางหลักของคุณ ไอคอนและป้ายกำกับแต่ละอันสอดคล้องกับพื้นที่ฟีเจอร์หลักของ AgencyHandy ต่อไปนี้คือสิ่งที่แต่ละส่วนทำ:

### CRM
ศูนย์กลางการจัดการความสัมพันธ์กับลูกค้าของคุณ จัดเก็บและจัดการข้อมูลติดต่อลูกค้าทั้งหมด ติดตามประวัติการสื่อสาร และรักษาบันทึกที่ชัดเจนของทุกบัญชีที่เอเจนซี่ของคุณทำงานด้วย ใช้ CRM เป็นแหล่งข้อมูลเดียวที่น่าเชื่อถือสำหรับว่าลูกค้าของคุณคือใครและความสัมพันธ์แต่ละอย่างอยู่ที่ไหน

### บริการ
กำหนดบริการที่เอเจนซี่ของคุณนำเสนอและสร้างแคตตาล็อกบริการ เมื่อลูกค้าสั่งซื้อ พวกเขาจะเลือกจากบริการที่คุณเผยแพร่ที่นี่ คุณสามารถตั้งราคา คำอธิบาย ผลงานที่ส่งมอบ และฟอร์มการรับคำสั่งซื้อสำหรับแต่ละบริการได้

### คำสั่งซื้อ
ติดตามคำสั่งซื้อของลูกค้าทุกรายการตั้งแต่การสั่งซื้อจนถึงการเสร็จสิ้น ส่วนคำสั่งซื้อให้คุณเห็นสถานะคำสั่งซื้อ งานที่เกี่ยวข้อง การสื่อสารกับลูกค้า และไทม์ไลน์การส่งมอบ — เพื่อไม่ให้สิ่งใดหลุดลอดระหว่างการขายและการส่งมอบ

### ข้อเสนอ
สร้าง ส่ง และจัดการข้อเสนอและสัญญาโดยตรงใน AgencyHandy ร่างข้อเสนอ แนบราคาและขอบเขต และส่งให้ลูกค้าตรวจสอบ เมื่อได้รับการยอมรับแล้ว ข้อเสนอสามารถแปลงเป็นคำสั่งซื้อที่ใช้งานอยู่โดยไม่ต้องทำซ้ำด้วยตนเอง

### ใบแจ้งหนี้
สร้างใบแจ้งหนี้แบบมืออาชีพที่เชื่อมโยงกับคำสั่งซื้อหรือโปรเจกต์ของลูกค้า ติดตามสถานะการชำระเงิน และให้ลูกค้าเห็นชัดเจนว่าพวกเขาค้างชำระเท่าไหร่ AgencyHandy จัดการวงจรชีวิตการออกใบแจ้งหนี้ตั้งแต่การสร้างจนถึงการชำระเสร็จสิ้น ลดการสื่อสารไปมาที่มักเกิดขึ้นกับการเรียกเก็บเงิน

### งาน
จัดการงานทั้งหมดในระดับงานย่อย สร้างงาน มอบหมายให้สมาชิกในทีม ตั้งวันครบกำหนดและลำดับความสำคัญ และติดตามสถานะการเสร็จสิ้น งานเชื่อมต่อโดยตรงกับคำสั่งซื้อและบริการ ดังนั้นความคืบหน้าจึงมองเห็นได้เสมอในบริบท

### ทีม
จัดการสมาชิกในทีมของคุณจากตำแหน่งศูนย์กลาง เชิญสมาชิกใหม่ มอบหมายบทบาท และควบคุมสิ่งที่แต่ละคนสามารถเข้าถึงได้ภายในเวิร์กสเปซ ใช้ส่วนนี้เพื่อให้รายชื่อทีมของคุณทันสมัยเมื่อเอเจนซี่เติบโต

### ทิกเก็ต
จัดการคำขอสนับสนุนของลูกค้าและการติดตามปัญหาภายในผ่านระบบ ticketing ในตัว ลูกค้าสามารถส่งทิกเก็ตจากพอร์ทัลของพวกเขา และทีมของคุณสามารถจัดเรียง ตอบสนอง และแก้ไขได้โดยไม่ต้องสลับไปใช้เครื่องมือ helpdesk แยกต่างหาก

### ตารางเวลาทำงาน
ติดตามเวลาที่ใช้ในงานลูกค้าและโปรเจกต์ภายใน สมาชิกในทีมสามารถบันทึกชั่วโมงกับงานหรือคำสั่งซื้อเฉพาะ ให้ข้อมูลที่แม่นยำสำหรับการเรียกเก็บเงิน การวางแผนกำลังคน และการรายงานความสามารถทำกำไรของโปรเจกต์

### ฟอร์ม
สร้างและจัดการฟอร์มที่กำหนดเองเพื่อรวบรวมข้อมูลที่มีโครงสร้างจากลูกค้าระหว่างการเริ่มต้นใช้งาน การรับคำสั่งซื้อ หรือจุดอื่นใดในขั้นตอนการทำงานของคุณ ฟอร์มที่สร้างที่นี่สามารถแนบกับบริการ ข้อเสนอ หรือส่งตรงถึงลูกค้า

### ไฟล์
จัดเก็บและจัดระเบียบไฟล์ทั้งหมดที่เกี่ยวข้องกับงานของเอเจนซี่ — ผลงานที่ส่งมอบ สัญญา ทรัพย์สินแบรนด์ และเอกสารลูกค้า — ในตำแหน่งรวมศูนย์เดียว ไฟล์สามารถกำหนดขอบเขตให้กับลูกค้าหรือคำสั่งซื้อเฉพาะเพื่อให้ทุกคนรู้ว่าจะหาสิ่งที่ต้องการได้ที่ไหน

### การตั้งค่า
กำหนดค่าทุกด้านของเวิร์กสเปซ AgencyHandy ของคุณ จัดการโปรไฟล์เอเจนซี่ การสร้างแบรนด์ การตั้งค่าการแจ้งเตือน การผสานรวม สิทธิ์ของทีม การเรียกเก็บเงิน และอื่น ๆ การตั้งค่าเป็นที่ที่คุณจะพบการกำหนดค่าระดับเวิร์กสเปซที่ใช้กับลูกค้าและสมาชิกในทีมทั้งหมด

## อ้างอิงด่วน: ส่วนแถบด้านข้างโดยสรุป

| ส่วน | การใช้งานหลัก |
|---|---|
| **CRM** | จัดการผู้ติดต่อลูกค้าและบันทึกบัญชี |
| **บริการ** | กำหนดและเผยแพร่แคตตาล็อกบริการของเอเจนซี่ |
| **คำสั่งซื้อ** | ติดตามและส่งมอบคำสั่งซื้อลูกค้าแบบครบวงจร |
| **ข้อเสนอ** | สร้าง ส่ง และจัดการข้อเสนอและสัญญา |
| **ใบแจ้งหนี้** | สร้างใบแจ้งหนี้และติดตามสถานะการชำระเงิน |
| **งาน** | มอบหมายงาน ตั้งกำหนดเวลา และติดตามการเสร็จสิ้น |
| **ทีม** | เชิญสมาชิก มอบหมายบทบาท ควบคุมการเข้าถึง |
| **ทิกเก็ต** | จัดการคำขอสนับสนุนลูกค้าและปัญหาภายใน |
| **ตารางเวลาทำงาน** | บันทึกและตรวจสอบเวลาที่ใช้ในงาน |
| **ฟอร์ม** | สร้างฟอร์มที่กำหนดเองสำหรับการรวบรวมข้อมูลลูกค้า |
| **ไฟล์** | รวมศูนย์เอกสารและผลงานที่ส่งมอบ |
| **การตั้งค่า** | กำหนดค่าเวิร์กสเปซ การสร้างแบรนด์ และสิทธิ์ |
