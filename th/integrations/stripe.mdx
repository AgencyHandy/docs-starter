การรวม Stripe ของ AgencyHandy ช่วยให้คุณรับการชำระเงินด้วยบัตรโดยตรงจากลูกค้าผ่านใบแจ้งหนี้ที่คุณส่งภายในแพลตฟอร์ม เมื่อเชื่อมต่อแล้ว ลูกค้าของคุณจะเห็นหน้าชำระเงินที่โฮสต์โดย Stripe ที่คุ้นเคย และสถานะการชำระเงินจะซิงค์กลับไปยัง AgencyHandy โดยอัตโนมัติ — ไม่จำเป็นต้องกระทบยอดด้วยตนเอง

<Note>
  เฉพาะบทบาท **SuperAdmin** และ **Admin** เท่านั้นที่สามารถเชื่อมต่อหรือจัดการการรวม Stripe ได้ คุณต้องมีบัญชี Stripe ที่ได้รับการยืนยันพร้อมสิทธิ์ที่เหมาะสมก่อนเริ่มต้น
</Note>

## เชื่อมต่อ Stripe

<Steps>
  <Step title="เปิดการตั้งค่าการรวม Stripe">
    ในแถบเมนูด้านซ้าย ไปที่ **Integrations → Stripe Integration**
  </Step>

  <Step title="อนุญาตบน Stripe">
    คลิก **Connect Stripe** คุณจะถูกเปลี่ยนเส้นทางไปยังหน้าการอนุญาตของ Stripe ลงชื่อเข้าใช้บัญชี Stripe ของคุณและคลิก **Allow** เพื่อให้สิทธิ์ AgencyHandy เข้าถึง เมื่ออนุมัติแล้ว Stripe จะเปลี่ยนเส้นทางคุณกลับไปยัง AgencyHandy โดยอัตโนมัติ
  </Step>

  <Step title="กำหนดค่าการตั้งค่าการชำระเงิน">
    หลังจากเปลี่ยนเส้นทางแล้ว กำหนดค่าตัวเลือกต่อไปนี้:

    - **Currency** — เลือกสกุลเงินการเรียกเก็บเงินเริ่มต้นสำหรับ workspace ของคุณ
    - **วิธีการชำระเงิน** — เปิดหรือปิดใช้งานประเภทบัตร (Visa, Mastercard ฯลฯ) ตามที่บัญชี Stripe ของคุณรองรับ
    - **การตั้งค่าการเรียกเก็บเงิน** — เลือกตั้งอัตราภาษี คำนำหน้าใบแจ้งหนี้ หรือเงื่อนไขการชำระเงินและวันครบกำหนดเริ่มต้น
  </Step>

  <Step title="บันทึกและยืนยัน">
    คลิก **Save** แบนเนอร์การยืนยันจะปรากฏขึ้นเมื่อการรวมเปิดใช้งาน ตอนนี้ Stripe เชื่อมโยงกับ workspace ของคุณแล้วและลูกค้าสามารถชำระใบแจ้งหนี้ออนไลน์ได้
  </Step>
</Steps>

## ทดสอบการรวม

ก่อนส่งใบแจ้งหนี้จริง ให้ทดสอบแบบ end-to-end อย่างรวดเร็วเพื่อยืนยันว่าการเชื่อมต่อทำงานได้อย่างถูกต้อง

<Steps>
  <Step title="สร้างใบแจ้งหนี้ทดสอบ">
    สร้างใบแจ้งหนี้ฉบับร่างภายใน AgencyHandy และกำหนดจำนวนเงินเล็กน้อย
  </Step>
  <Step title="ส่งหาตัวเอง">
    ส่งใบแจ้งหนี้ไปยังที่อยู่อีเมลของตัวเองหรือสมาชิกทีมที่สามารถช่วยในการทดสอบ
  </Step>
  <Step title="ทำการชำระเงินทดสอบ">
    เปิดลิงก์ใบแจ้งหนี้และทำการชำระเงินโดยใช้บัตรทดสอบ Stripe (เช่น `4242 4242 4242 4242`) หากบัญชี Stripe ของคุณอยู่ในโหมดทดสอบ หรือบัตรจริงในโหมด live
  </Step>
  <Step title="ตรวจสอบการอัปเดตสถานะ">
    กลับไปที่ AgencyHandy และยืนยันว่าสถานะใบแจ้งหนี้เปลี่ยนเป็น **Paid** หากเปลี่ยนแล้ว การรวมของคุณทำงานได้อย่างถูกต้อง
  </Step>
</Steps>

## จัดการการเชื่อมต่อ Stripe ของคุณ

กลับไปที่ **Integrations → Stripe Integration** ได้ตลอดเวลาเพื่อ:

- **อัปเดตการตั้งค่าการชำระเงิน** — เปลี่ยนสกุลเงิน วิธีการชำระเงิน หรือการตั้งค่าการเรียกเก็บเงิน
- **ตรวจสอบสถานะการเชื่อมต่อ** — ตรวจสอบว่า Stripe เปิดใช้งานอยู่หรือไม่
- **ยกเลิกการเชื่อมต่อ** — คลิก **Disconnect** เพื่อยกเลิกการเชื่อมโยงบัญชี Stripe ของคุณ การรวบรวมการชำระเงินจะหยุดทันทีจนกว่าคุณจะเชื่อมต่อใหม่
- **เชื่อมต่อใหม่** — คลิก **Connect Stripe** อีกครั้งและให้อนุญาตอีกครั้งเพื่อกู้คืนการรวม

<Warning>
  การยกเลิกการเชื่อมต่อ Stripe จะหยุดการรวบรวมการชำระเงินสำหรับใบแจ้งหนี้ที่ค้างอยู่ทั้งหมด ลูกค้าจะไม่สามารถชำระเงินออนไลน์ได้จนกว่าคุณจะเชื่อมต่อใหม่และให้อนุญาตอีกครั้ง
</Warning>

## หมายเหตุเพิ่มเติม

<Note>
  AgencyHandy ไม่เก็บข้อมูลการชำระเงินของคุณ ข้อมูลการชำระเงินทั้งหมดถูกจัดการโดยโครงสร้างพื้นฐานที่ปลอดภัยของ Stripe เท่านั้น
</Note>

- **บัญชีเดียวต่อ workspace** — คุณสามารถเชื่อมต่อบัญชี Stripe เดียวกับแต่ละ workspace ของ AgencyHandy ในการสลับบัญชี ให้ยกเลิกการเชื่อมต่อบัญชีปัจจุบันก่อน จากนั้นเชื่อมต่อใหม่กับบัญชีใหม่
- **สถานะใบแจ้งหนี้แบบ live** — ใบแจ้งหนี้ทั้งหมดที่เชื่อมโยงกับ Stripe ยังคงสะท้อนการอัปเดตสถานะแบบ real-time ที่ได้รับโดยตรงจาก Stripe
- **สกุลเงิน** — ตรวจสอบให้แน่ใจว่าสกุลเงินที่กำหนดค่าใน AgencyHandy ตรงกับสกุลเงินที่เปิดใช้งานในบัญชี Stripe ของคุณเพื่อหลีกเลี่ยงข้อผิดพลาดในการชำระเงิน
