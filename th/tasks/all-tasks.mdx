หน้างานทั้งหมดใน AgencyHandy มอบมุมมองแบบรวมศูนย์เดียวของทุกงานที่อยู่ในขั้นตอนการทำงานที่ใช้งานอยู่ทั่วทั้งเอเจนซีของคุณ แทนที่จะค้นหาในออเดอร์แต่ละรายการ คุณสามารถดูงานที่เกี่ยวข้องทั้งหมดในที่เดียว กรองตามผู้รับมอบหมาย ลำดับความสำคัญ หรือวันที่ และแก้ไขได้โดยตรงจากรายการ การมองเห็นถูกกำหนดขอบเขตโดยอัตโนมัติตามบทบาทของคุณ Super Admin เห็นทุกอย่าง PM เห็นออเดอร์ที่ได้รับมอบหมาย และ Employee เห็นเฉพาะงานของตนเอง

## การเข้าถึงหน้างานทั้งหมด

1. เข้าสู่ระบบบัญชี AgencyHandy ของคุณ
2. คลิก **Tasks** ในแถบด้านซ้าย
3. เลือก **All Tasks** เพื่อเปิดแดชบอร์ดงานแบบรวมศูนย์

<Note>
  หน้างานทั้งหมดจะแสดงเฉพาะงานจากออเดอร์ที่มีสถานะ **Ongoing** หรือ **In Review** งานที่ผูกกับออเดอร์ในสถานะอื่นๆ (เช่น Completed, Pending) จะไม่ปรากฏที่นี่
</Note>

## การมองเห็นงานตามบทบาท

สิ่งที่คุณเห็นในหน้างานทั้งหมดขึ้นอยู่กับบทบาทของคุณอย่างสมบูรณ์

| บทบาท | สิ่งที่พวกเขาเห็น |
|------|---------------|
| **Super Admin / Admin** | งานทั้งหมดข้ามทุกออเดอร์ ทีม และสถานะ |
| **Project Manager (PM)** | เฉพาะงานจากออเดอร์ที่พวกเขาได้รับมอบหมายเป็น Manager |
| **Employee / Assignee** | เฉพาะงานที่มอบหมายให้โดยตรง แม้เพื่อนร่วมทีมจะทำงานในออเดอร์เดียวกัน |

<Tip>
  หาก PM ไม่พบงานที่คาดว่าจะเห็น ให้ยืนยันว่าพวกเขาได้รับมอบหมายเป็น Manager ในออเดอร์ที่เกี่ยวข้อง PM จะไม่เห็นงานเว้นแต่จะเป็น Manager ที่ได้รับมอบหมายสำหรับออเดอร์นั้น
</Tip>

งานจะปรากฏในหน้างานทั้งหมดเมื่อเป็นไปตามเงื่อนไขเหล่านี้:

## การค้นหา กรอง และเรียงลำดับ

หน้างานทั้งหมดมีชุดตัวควบคุมครบถ้วนเพื่อช่วยให้คุณค้นหาสิ่งที่ต้องการได้อย่างแม่นยำ

### การค้นหา

ใช้แถบค้นหาเพื่อค้นหางานตามชื่อ ผู้รับมอบหมาย หรือออเดอร์ที่เกี่ยวข้อง ผลลัพธ์จะอัปเดตแบบเรียลไทม์ขณะที่คุณพิมพ์

### ตัวกรอง

คลิก **Filter → + Add Filter** เพื่อเปิดแผงตัวกรอง คุณสามารถรวมหลายตัวกรองเพื่อผลลัพธ์ที่แม่นยำ:

| ตัวกรอง | สิ่งที่ทำ |
|--------|--------------|
| **Order ID** | แสดงงานที่เชื่อมกับหมายเลขออเดอร์ที่ระบุ |
| **Assignee** | แสดงงานที่มอบหมายให้สมาชิกทีมคนใดคนหนึ่ง |
| **Status** | กรองตามความคืบหน้าของงาน: Pending, In Progress, Review, Done เป็นต้น |
| **Priority** | มุ่งเน้นงานที่มีลำดับความสำคัญสูง ปานกลาง หรือต่ำ |
| **Label** | จำกัดตามป้ายกำกับหรือหมวดหมู่ของงาน |
| **Kick Off Date** | แสดงงานที่เริ่มต้นในหรือหลังจากวันที่ระบุ |
| **End Date** | แสดงงานที่ครบกำหนดก่อนวันที่ระบุ |

<Note>
  ตัวกรองใช้ได้เฉพาะในหน้างานทั้งหมด ตัวกรองจะคงอยู่ในระหว่างเซสชันเบราว์เซอร์ของคุณแต่จะรีเซ็ตเมื่อโหลดหน้าใหม่ ตัวกรองมีผลต่อมุมมองของคุณเท่านั้นและไม่เปลี่ยนสิ่งที่ผู้ใช้อื่นเห็น
</Note>

หากต้องการล้างตัวกรองทั้งหมด คลิก **Clear Filters** เพื่อกลับสู่รายการที่ไม่มีการกรองแบบเต็ม

### การเรียงลำดับ

เรียงลำดับรายการงานตาม:
- **Due date** — แสดงงานที่กำลังจะถึงหรือเลยกำหนดแล้ว
- **Priority** — นำงานที่สำคัญที่สุดขึ้นมาด้านบน
- **Creation time** — ดูงานที่เพิ่มล่าสุดก่อน

### การนำทางแบบ Breadcrumb

ใช้เส้นทาง breadcrumb ที่ด้านบนของหน้าเพื่อรักษาบริบทขณะที่คุณนำทางระหว่างงานย่อย รายการงาน และออเดอร์หลัก สิ่งนี้มีประโยชน์อย่างยิ่งเมื่อเจาะลึกงานที่เฉพาะเจาะจงแล้วต้องกลับไปยังรายการที่กว้างกว่า

## การแก้ไขงาน

คุณสามารถแก้ไขงานในหน้างานทั้งหมดได้สองวิธี:

  </Tab>

  </Tab>

## กฎการลบงาน

งานในหน้างานทั้งหมดเชื่อมโยงโดยตรงกับออเดอร์หลักของตน

- เมื่อ **ออเดอร์ถูกลบ** งานทั้งหมดที่เชื่อมกับออเดอร์นั้นจะ **ถูกลบออกโดยอัตโนมัติและถาวร** จากหน้างานทั้งหมด
- การลบนี้มีผลต่อทุกบทบาทผู้ใช้ ไม่มีใครสามารถเห็นงานเหล่านั้นได้หลังจากออเดอร์ถูกลบ
- **ไม่มีตัวเลือกการกู้คืน** สำหรับงานที่ถูกลบผ่านการลบออเดอร์

<Warning>
  การลบออเดอร์จะลบงานทั้งหมดของออเดอร์นั้นอย่างถาวร ตรวจสอบให้แน่ใจว่าบันทึกประวัติงานหรือผลลัพธ์ที่สำคัญได้รับการบันทึกไว้ที่อื่นก่อนลบออเดอร์
</Warning>

## หมายเหตุเพิ่มเติม

<Accordion title="การอัปเดตแบบเรียลไทม์">
  การเปลี่ยนแปลงออเดอร์ การมอบหมายงาน หรือสถานะงานจะสะท้อนในหน้างานทั้งหมดแบบเรียลไทม์ หากเพื่อนร่วมทีมอัปเดตงานขณะที่คุณเปิดหน้านั้น คุณจะเห็นการเปลี่ยนแปลงโดยไม่ต้องโหลดหน้าใหม่
</Accordion>

<Accordion title="งานที่ลูกค้าร้องขอ">
  งานที่ลูกค้าส่งมาจะไม่ปรากฏในหน้างานทั้งหมดจนกว่าเอเจนซีจะยอมรับอย่างชัดเจน จนกว่าจะได้รับการยอมรับ งานเหล่านี้จะอยู่ในสถานะรอดำเนินการและไม่ปรากฏในมุมมองรวมศูนย์
</Accordion>

<Accordion title="งานและสถานะออเดอร์">
  งานจะปรากฏเฉพาะสำหรับออเดอร์ที่มีสถานะ Ongoing หรือ In Review หากออเดอร์ย้ายไปยัง Completed หรือถูกหยุดชั่วคราว งานของออเดอร์นั้นจะไม่แสดงที่นี่อีกต่อไป ไปที่ออเดอร์เฉพาะเพื่อดูงานที่ผูกกับออเดอร์นอกสถานะเหล่านี้
</Accordion>
