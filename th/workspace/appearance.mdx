การตั้งค่ารูปลักษณ์ให้คุณควบคุมอย่างเต็มที่ว่าเวิร์กสเปซ AgencyHandy ของคุณจะดูอย่างไร คุณสามารถปรับโทนสีของแพลตฟอร์มให้สอดคล้องกับเอกลักษณ์แบรนด์ของเอเจนซี่โดยการกำหนดสีหลัก สีพื้นหลังแถบด้านข้าง สีข้อความแถบด้านข้าง และสี hover highlight สำหรับรายการนำทางในแถบด้านข้าง การเปลี่ยนแปลงทุกอย่างที่คุณบันทึกจะมีผลทันทีทั่วทั้งเวิร์กสเปซสำหรับผู้ใช้ทุกคน — ไม่จำเป็นต้องรีเฟรชหน้า

<Note>
  เฉพาะ **Super Admin** และ **Admin** เท่านั้นที่สามารถเข้าถึงและแก้ไขการตั้งค่ารูปลักษณ์ได้ หากคุณไม่เห็นตัวเลือกนี้ใต้ **Workspace Config** ให้ขอให้ Super Admin ตรวจสอบบทบาทของคุณ
</Note>

## การเข้าถึงการตั้งค่ารูปลักษณ์

<Steps>
  <Step title="เปิด Workspace Config">
    จากเมนูนำทางซ้าย คลิก **Workspace Config**
  </Step>
  <Step title="เลือก Appearance">
    คลิก **Appearance** (บางครั้งมีป้ายกำกับว่า **Visual Settings**) จากเมนูย่อย
  </Step>
</Steps>

---

## การปรับแต่งองค์ประกอบภาพแต่ละอย่าง

### สีหลัก

สีหลักถูกนำไปใช้กับปุ่ม ลิงก์ที่ใช้งานอยู่ สถานะที่เน้น และองค์ประกอบอินเทอร์เฟซสำคัญทั่วทั้งแพลตฟอร์ม

- เลือกสีที่สะท้อนเอกลักษณ์แบรนด์หลักของคุณ
- นี่คือการตั้งค่าสีที่มีผลกระทบมากที่สุดเพราะปรากฏทุกที่ใน UI

### สีพื้นหลังแถบด้านข้าง

สีพื้นหลังแถบด้านข้างกำหนดเฉดสีพื้นฐานของแผงนำทางซ้าย

- ใช้สีที่สร้างการแยกภาพที่ชัดเจนระหว่างแถบด้านข้างและพื้นที่เนื้อหาหลัก
- พื้นหลังสีเข้มพร้อมข้อความสีอ่อนโดยทั่วไปให้ผลลัพธ์ที่ดีที่สุดสำหรับการอ่าน

### สีข้อความแถบด้านข้าง

สีข้อความแถบด้านข้างควบคุมสีป้ายกำกับสำหรับรายการนำทางทั้งหมดในเมนูซ้าย

- เลือกสีที่มีความเปรียบต่างสูงกับสีพื้นหลังแถบด้านข้างของคุณเสมอ
- ความเปรียบต่างที่ต่ำทำให้ลิงก์อ่านยากขึ้นและลดการเข้าถึง

### สี Hover พื้นหลังแถบด้านข้าง

สีนี้ปรากฏเป็น highlight พื้นหลังเมื่อผู้ใช้เลื่อนเคอร์เซอร์ไปยังรายการนำทางในแถบด้านข้าง

- สัญญาณถึงการโต้ตอบและช่วยให้ผู้ใช้เข้าใจว่ารายการใดที่พวกเขากำลังจะคลิก
- เลือกสีที่แตกต่างอย่างเห็นได้ชัดจากทั้งสีพื้นหลังแถบด้านข้างเริ่มต้นและสีรายการที่ใช้งานอยู่

## การนำสีที่เลือกไปใช้

<Steps>
  <Step title="เปิดตัวเลือกสีสำหรับแต่ละฟิลด์">
    คลิกตัวอย่างสีถัดจากฟิลด์รูปลักษณ์สี่อย่าง — Primary Color, Sidebar Background, Sidebar Text Color หรือ Sidebar Hover Background Color — เพื่อเปิดตัวเลือกสี
  </Step>
  <Step title="กรอก hex code หรือเลือกสี">
    พิมพ์ hex code ที่แน่นอนของแบรนด์ (เช่น `#3B5BDB`) โดยตรงในฟิลด์ input หรือใช้ตัวเลือกภาพเพื่อเลือกเฉด
  </Step>
  <Step title="ดูตัวอย่างการเปลี่ยนแปลง">
    ตรวจสอบว่าสีที่อัปเดตดูอย่างไรในแถบด้านข้างและทั่วทั้งอินเทอร์เฟซก่อนบันทึก
  </Step>
  <Step title="คลิก Save Changes">
    เมื่อคุณพอใจกับผลลัพธ์แล้ว คลิก **Save Changes** รูปลักษณ์ที่อัปเดตจะถูกนำไปใช้ทั่วทั้งระบบและทันทีสำหรับผู้ใช้ทุกคนในเวิร์กสเปซ
  </Step>
</Steps>

<Tip>
  เก็บบันทึก hex color codes ของแบรนด์ไว้ในที่ที่เข้าถึงได้ (คู่มือสไตล์แบรนด์หรือไฟล์บันทึก) คุณจะต้องนำค่าที่แน่นอนเดียวกันมาใช้ซ้ำอย่างสม่ำเสมอ — โดยเฉพาะอย่างยิ่งหากคุณต้องรีเซ็ตและกำหนดค่าการตั้งค่ารูปลักษณ์ใหม่
</Tip>

---

## การรีเซ็ตเป็นธีมเริ่มต้น

หากคุณต้องการลบการสร้างแบรนด์ทั้งหมดและคืนค่าโทนสีเริ่มต้นของ AgencyHandy คลิกปุ่ม **Reset Default Settings** บนหน้า Appearance

<Warning>
  การรีเซ็ตเป็นค่าเริ่มต้นจะลบการเปลี่ยนแปลงสีที่กำหนดเองทั้งหมดอย่างถาวร การกระทำนี้ไม่สามารถยกเลิกได้ หากคุณต้องการรักษาการตั้งค่าปัจจุบัน ให้จดบันทึก hex codes ก่อนรีเซ็ต
</Warning>

  </Card>

## หมายเหตุสำคัญ

- **ผลกระทบทั่วทั้งระบบทันที:** การเปลี่ยนแปลงรูปลักษณ์ที่บันทึกทั้งหมดจะถูกนำไปใช้ทันทีทั่วทั้งเวิร์กสเปซ ผู้ใช้ที่ล็อกอินทุกคนจะเห็นการอัปเดตในการดำเนินการนำทางครั้งถัดไป
- **ธีมเดียวที่ใช้งานได้:** ไม่มีการรองรับหลายธีม — ชุดสีหนึ่งชุดใช้กับเวิร์กสเปซทั้งหมดสำหรับทุกบทบาท
- **ข้อจำกัดบทบาท:** เฉพาะ Super Admin และ Admin เท่านั้นที่สามารถเปลี่ยนแปลงรูปลักษณ์ได้ สมาชิกทีมปกติและลูกค้าไม่สามารถแก้ไขสีได้
- **การรีเซ็ตเป็นการถาวร:** ปุ่ม **Reset Default Settings** ไม่สามารถยกเลิกได้ ดังนั้นใช้เฉพาะเมื่อคุณตั้งใจจะลบสีที่กำหนดเองทั้งหมด
