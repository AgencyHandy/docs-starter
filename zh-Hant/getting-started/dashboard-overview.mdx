登入後，AgencyHandy 會帶您進入主儀表板——這是您代理商管理一切事務的控制中心。介面清晰且以導覽為主：持久性的左側邊欄讓您即時存取每個主要功能，而中央面板則會更新以顯示您所選區段的相關資料。從客戶關係到發票到任務追蹤——經營代理商所需的一切都在這裡。

## 兩種儀表板視圖

AgencyHandy 根據登入者的身份提供兩種不同的儀表板視角。您可以按需切換。

<CardGroup cols={2}>
  <Card title="代理商儀表板" icon="building">
    代理商擁有者、管理員和團隊成員的預設視圖。使用它從單一中央視圖即時掌握代理商的營運狀況——包括有效訂單、未結工單、未付發票、任務進度和團隊工作負載。這是您大部分時間所在的儀表板。
  </Card>
  <Card title="客戶儀表板" icon="user">
    專為客戶登入 AgencyHandy 時存取而設計的入口網站。客戶可以監控有效訂單、查看已指派的任務、查看未付發票，以及提交支援工單——讓他們對您代表他們交付的工作擁有完全的透明度。
  </Card>
</CardGroup>

<Note>
  代理商儀表板和客戶儀表板顯示相同的底層資料，但視角不同且權限等級也不同。客戶只能看到與其自身帳戶相關的資料；代理商用戶可以看到所有客戶的資料。
</Note>

## 左側邊欄導覽

左側邊欄是您的主要導覽面板。每個圖示和標籤對應 AgencyHandy 的一個主要功能區域。以下是各個區段的功能說明：

### CRM
您的客戶關係管理中心。儲存和管理所有客戶聯絡資料、追蹤溝通歷程，並維護代理商所服務的每個帳戶的清晰記錄。將 CRM 作為了解客戶是誰以及每段關係進展狀況的唯一可信來源。

### 服務
定義代理商提供的服務並建立您的服務目錄。當客戶下訂單時，他們從您在此發布的服務中進行選擇。您可以為每項服務設定定價、描述、交付成果和訂單接受表單。

### 訂單
追蹤每個客戶訂單從下單到完成的全過程。訂單區段讓您清楚了解訂單狀態、相關任務、客戶溝通和交付時間表——確保從銷售到交付之間沒有任何遺漏。

### 提案
直接在 AgencyHandy 內建立、發送和管理提案與合約。起草提案、附上定價和範疇，並發送給客戶審核。一旦接受，提案可以轉換為有效訂單，無需任何手動複製。

### 發票
產生與客戶訂單或專案掛鉤的專業發票、追蹤付款狀態，並讓客戶清楚了解其應付金額。AgencyHandy 處理從建立到結清的整個發票生命週期，減少帳單通常伴隨的來回往返。

### 任務
在任務層級管理所有工作。建立任務、分配給團隊成員、設定截止日期和優先順序，並監控完成狀態。任務直接連結到訂單和服務，因此進度始終在上下文中可見。

### 團隊
從中央位置管理您的團隊成員。邀請新成員、分配角色，並控制每個人在工作區中的存取權限。隨著代理商的成長，使用此區段保持團隊名冊的最新狀態。

### 工單
透過內建工單系統處理客戶支援請求和內部問題追蹤。客戶可以從其入口網站提交工單，您的團隊可以對其進行分類、回應和解決，無需切換到獨立的客服工具。

### 工時表
追蹤花在客戶工作和內部專案上的時間。團隊成員可以針對特定任務或訂單記錄工時，為您提供準確的帳單、產能規劃和專案獲利報告資料。

### 表單
建立和管理自訂表單，以在客戶引導、訂單接受或工作流程中任何其他節點收集客戶的結構化資訊。在此建立的表單可以附加到服務、提案，或直接發送給客戶。

### 檔案
將與代理商工作相關的所有檔案——交付成果、合約、品牌素材和客戶文件——集中儲存和整理在同一地點。檔案可以限定範圍到特定客戶或訂單，讓所有人隨時知道在哪裡找到所需內容。

### 設定
設定 AgencyHandy 工作區的每個方面。管理代理商個人資料、品牌、通知偏好、整合、團隊權限、帳單等。設定也是您找到適用於所有客戶和團隊成員的工作區層級設定的地方。

## 快速參考：邊欄區段一覽

| 區段 | 主要用途 |
|---|---|
| **CRM** | 管理客戶聯絡資訊和帳戶記錄 |
| **服務** | 定義並發布代理商的服務目錄 |
| **訂單** | 端對端追蹤和交付客戶訂單 |
| **提案** | 建立、發送和管理提案與合約 |
| **發票** | 產生發票並監控付款狀態 |
| **任務** | 分配工作、設定截止日期並追蹤完成情況 |
| **團隊** | 邀請成員、分配角色、控制存取權限 |
| **工單** | 處理客戶支援請求和內部問題 |
| **工時表** | 記錄和查看工作花費的時間 |
| **表單** | 為客戶資料收集建立自訂表單 |
| **檔案** | 集中管理文件和交付成果 |
| **設定** | 設定工作區、品牌和權限 |
