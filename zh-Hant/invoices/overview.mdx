AgencyHandy 的發票功能為您的代理商提供一個完整的帳單和付款管理系統，與您用於管理客戶、訂單和專案的平台整合在一起。您可以建立專業發票、透過電子郵件發送給客戶、即時追蹤付款狀態、為訂閱服務自動化定期帳單，以及隨時匯出財務記錄。每張發票都會自動附上您代理商的標誌和地址，確保您的通訊保持一致的品牌形象。

## 誰可以建立發票

只有具備以下角色的使用者才能建立和管理發票：

- **超級管理員**
- **管理員**
- **專案經理（PM）**

客戶可以查看和下載發票，但無法建立或編輯發票。

## 發票儀表板

若要開啟發票儀表板，請點擊左側側邊欄中的 **發票**。您可以在此：

- 建立、編輯和刪除發票
- 按 **客戶名稱**、**訂單 ID** 或 **服務名稱** 搜尋發票
- 篩選和排序發票清單
- 一眼查看發票狀態
- 存取發票設定，包括前綴配置

## 發票狀態

每張發票都會經歷一個定義好的生命週期。了解這些狀態有助於您有效管理應收帳款。

| 狀態 | 說明 |
|--------|-------------|
| **草稿** | 發票已建立但尚未完成。您可以編輯其中任何部分。準備好後，將狀態變更為開放以完成發票。未與訂閱關聯的草稿發票可以刪除。 |
| **開放／未付款** | 發票已完成，等待付款。代理商將此發送給客戶，客戶可以查看並付款。您可以將開放發票變更為已付款、已作廢或無法收回。 |
| **已付款** | 發票已付款。當您將發票標記為已付款時，系統會自動記錄付款日期。 |
| **已作廢** | 發票已取消。系統自動記錄作廢日期。 |
| **無法收回** | 客戶不太可能付款。在您的會計流程中將其視為壞帳處理。 |

<Note>
  當您使用 **發送給客戶** 按鈕將發票發送給客戶時，狀態會自動從 **草稿** 變更為 **開放**。只有代理商才能將狀態從開放變更為已付款、已作廢或無法收回。
</Note>

## 直接發票（不含訂單 ID）

AgencyHandy 支援 **直接發票**——未連結至任何現有訂單的發票。這對於一次性帳單、臨時付款，或向尚未在您系統中註冊為客戶的聯絡人開票非常有用。

直接發票的主要功能：

- 無需選擇訂單 ID 即可發送發票
- 向任何現有客戶開票，或使用 **獨立電子郵件** 向任意電子郵件地址發送
- 手動新增項目、稅金和折扣
- 立即發送或儲存為草稿

<Tip>
  當您需要為 AgencyHandy 中沒有正式訂單的工作快速向客戶收費時，請使用直接發票，例如緊急專案、諮詢通話或範圍外的工作。
</Tip>

## 匯出發票

您可以匯出發票清單，用於外部會計工具或記錄保存。匯出內容包含所有可見欄位：發票 ID、客戶名稱、服務、金額、狀態及日期。請從發票儀表板存取匯出選項。

## 發票設定

從發票儀表板右上角的 **設定** 按鈕配置全域發票設定。包括：

- **發票前綴** — 自訂每個發票編號前出現的前綴
- **提醒範本** — 為未付款發票設定自動付款提醒
