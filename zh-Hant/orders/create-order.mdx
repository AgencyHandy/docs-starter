在 AgencyHandy 中建立訂單，可為您提供一個有條理的工作區，用以向客戶交付服務。您可預先填入關鍵資訊——客戶、服務、定價、日期及專案說明——其他所有內容，從任務到討論，都將圍繞這筆單一記錄展開。本頁面將引導您從頭建立訂單、事後更新、變更狀態，以及在不再需要時刪除訂單。

## 需求條件

- 您必須已登入 AgencyHandy。
- 您必須擁有 **超級管理員**、**管理員** 或 **專案經理** 角色，才能建立或編輯訂單。
- 只有 **超級管理員** 和 **管理員** 可以刪除訂單。

## 建立新訂單

<Steps>
  <Step title="前往訂單頁面">
    在 AgencyHandy 儀表板的左側側邊欄中，點擊 **訂單**。
  </Step>

  <Step title="開啟建立表單">
    點擊右上角的 **+ 建立新訂單**，表單將會出現。
  </Step>

  <Step title="填寫訂單詳情">
    填寫表單中的各個欄位：

    | 欄位 | 必填 | 備註 |
    |---|---|---|
    | **客戶** | ✅ | 從現有客戶清單中選擇 |
    | **服務** | ✅ | 包含已發布及未發布的服務 |
    | **專案經理** | ❌ | 選填；指派一位 PM 負責交付工作 |
    | **數量** | ✅ | 訂購的單位數量 |
    | **價格** | ✅ | 訂單的議定預算 |
    | **啟動日期** | ✅ | 工作開始的日期 |
    | **截止日期** | ✅ | 交付期限 |
    | **訂單說明** | ❌ | 提供給團隊的說明、需求或參考備注 |
    | **附件** | ❌ | 與訂單相關的支援文件 |

    <Tip>
      若所選服務已連結附加項目，表單中將出現 **附加項目** 區塊。請在提交前加入所有相關的附加項目。
    </Tip>
  </Step>

  <Step title="確認填寫內容">
    再次確認所有欄位的正確性——尤其是客戶、服務、價格及截止日期。
  </Step>

  <Step title="提交訂單">
    點擊 **建立** 以完成訂單。訂單建立成功後，您將收到確認訊息。
  </Step>
</Steps>

### 建立訂單後的後續動作

- 公司中所有超級管理員將自動獲得新訂單的存取權限。
- 客戶將收到其訂單已建立的通知。
- 訂單將以 **待處理** 狀態顯示於 **訂單** 清單中。

## 編輯或更新訂單

您可以隨時更新訂單，以修正資訊、調整定價、更改截止日期或配合客戶需求。

<Note>
您必須是 **超級管理員**、**管理員** 或 **專案經理** 才能編輯訂單。
</Note>

<Steps>
  <Step title="開啟訂單清單">
    在左側側邊欄中前往 **訂單**。
  </Step>

  <Step title="選取要編輯的訂單">
    找到您要更新的訂單，並勾選其旁邊的核取方塊。
  </Step>

  <Step title="啟用編輯按鈕">
    右上角將出現 **編輯** 按鈕。點擊它以在編輯模式下開啟訂單詳情。
  </Step>

  <Step title="修改必要欄位">
    更新需要變更的欄位——客戶資訊、服務、定價、日期、說明或指派的 PM。
  </Step>

  <Step title="儲存變更">
    點擊 **更新** 以儲存。確認訊息將確認更新成功。
  </Step>
</Steps>

## 變更訂單狀態

訂單狀態反映了工作在交付流程中所處的階段。AgencyHandy 定義了五種固定狀態——您無法新增或重新命名它們。

| 狀態 | 含義 |
|---|---|
| **待處理** | 已建立，尚未開始 |
| **進行中** | 正在積極處理 |
| **審核中** | 已提交審核；等待驗收 |
| **已完成** | 已交付並獲得驗收 |
| **已取消** | 不再有效 |

<Note>
只有 **超級管理員**、**管理員** 和 **專案經理** 可以變更訂單狀態。員工和客戶無法變更狀態。
</Note>

若要更新狀態，請開啟訂單，並使用訂單詳情頁面頂部的狀態選擇器。從下拉選單中選擇新狀態並確認。

## 刪除訂單

刪除訂單會將其從系統中永久移除。請僅在訂單確實已過時時才使用此功能——例如，若訂單是錯誤建立或重複建立的。

<Warning>
訂單刪除操作不可逆。一旦刪除訂單，便無法還原。請謹慎操作。
</Warning>

<Note>
只有 **超級管理員** 和 **管理員** 可以刪除訂單。
</Note>

<Steps>
  <Step title="前往訂單頁面">
    點擊左側側邊欄中的 **訂單**。
  </Step>

  <Step title="尋找要刪除的訂單">
    在清單中捲動或搜尋您要移除的訂單。
  </Step>

  <Step title="啟動刪除操作">
    點擊與該訂單關聯的 **垃圾桶圖示**。
  </Step>

  <Step title="確認刪除">
    將出現確認提示。確認後即永久刪除訂單。
  </Step>
</Steps>

刪除後，請檢視剩餘訂單以確保一切正常。已刪除的記錄將不再出現於任何清單或報表中。

## 在訂單中新增團隊成員

超級管理員、管理員和 PM 可以隨時將員工新增至訂單。當團隊成員被新增時，他們將收到應用程式內通知。被指派的員工可以查看訂單及其任務，但他們無法管理訂單本身（編輯詳情、變更狀態或刪除訂單）。
