AgencyHandy 中的「所有任務」頁面為您提供代理商中所有有效工作流程中每項任務的單一集中檢視。您無需在個別訂單中逐一尋找，而是可以在一個地方看到所有相關任務，按負責人、優先級或日期進行篩選，並直接從清單中進行編輯。能見度會根據您的角色自動限定範圍——超級管理員可以看到所有內容，專案經理可以看到其指派的訂單，而員工只能看到自己的任務。

## 存取「所有任務」頁面

1. 登入您的 AgencyHandy 帳戶。
2. 點擊左側側邊欄中的 **任務**。
3. 選擇 **所有任務** 以開啟集中式任務儀表板。

<Note>
  「所有任務」頁面只顯示狀態為 **進行中** 或 **審核中** 的訂單中的任務。與其他狀態訂單（例如：已完成、待處理）相關的任務不會顯示在此處。
</Note>

## 依角色的任務能見度

您在「所有任務」頁面上看到的內容完全取決於您的角色。

| 角色 | 他們看到的內容 |
|------|---------------|
| **超級管理員 / 管理員** | 跨每個訂單、團隊和狀態的所有任務 |
| **專案經理（PM）** | 僅限他們被指派為負責人的訂單中的任務 |
| **員工 / 被指派人** | 僅限直接指派給他們的任務——即使隊友正在同一訂單中工作 |

<Tip>
  如果 PM 找不到他們期望看到的任務，請確認他們是否被指派為相關訂單的負責人。PM 除非是該訂單的指派負責人，否則不會看到任務。
</Tip>

## 任務何時出現在此頁面

當滿足以下條件時，任務會出現在「所有任務」頁面上：

<CardGroup cols={2}>
  <Card title="觸發任務能見度的訂單" icon="circle-check">
    - 手動訂單（一次性或訂閱）已建立
    - 透過 Stripe 或手動付款方式下達的目錄訂單
    - 提案已被代理商接受
  </Card>
  <Card title="使任務保持隱藏的訂單" icon="eye-slash">
    - 代理商尚未接受的客戶請求任務
    - 代理商尚未接受的試用訂單
    - 等待手動銀行付款確認的待處理狀態訂單
  </Card>
</CardGroup>

## 搜尋、篩選及排序

「所有任務」頁面包含一套完整的控制項，幫助您精確找到所需內容。

### 搜尋

使用搜尋列按名稱、負責人或相關訂單查找任務。結果在您輸入時即時更新。

### 篩選器

點擊 **篩選器 → + 新增篩選器** 以開啟篩選面板。您可以組合多個篩選器以獲得精確結果：

| 篩選器 | 功能說明 |
|--------|--------------|
| **訂單 ID** | 顯示連結至特定訂單編號的任務 |
| **負責人** | 顯示指派給特定團隊成員的任務 |
| **狀態** | 按任務進度篩選：待處理、進行中、審核中、已完成等 |
| **優先級** | 專注於高、中或低優先級任務 |
| **標籤** | 依任務標籤或類別縮小範圍 |
| **啟動日期** | 顯示在特定日期當天或之後開始的任務 |
| **結束日期** | 顯示在特定日期前到期的任務 |

<Note>
  篩選器僅在「所有任務」頁面上可用。它們在您的瀏覽器工作階段期間持續有效，但在頁面重新整理時重置。篩選器只影響您的檢視，不會改變其他使用者看到的內容。
</Note>

若要清除所有篩選器，請點擊 **清除篩選器** 以返回完整的未篩選清單。

### 排序

按以下方式排序任務清單：
- **截止日期** — 顯示即將到期或已逾期的任務
- **優先級** — 將最關鍵的任務置於頂部
- **建立時間** — 最先顯示最新新增的任務

### 麵包屑導覽

使用頁面頂部的麵包屑路徑，在子任務、任務清單和上層訂單之間導覽時保持脈絡。當您深入查看特定任務然後需要返回更廣泛的清單時，這特別有用。

## 編輯任務

您可以透過兩種方式在「所有任務」頁面上修改任務：

<Tabs>
  <Tab title="行內編輯">
    直接點擊任務列中的可編輯欄位——例如狀態、負責人或截止日期——無需離開清單檢視即可進行更改。這是快速更新的最快選項。
  </Tab>
  <Tab title="視窗編輯">
    點擊任務名稱以開啟完整的任務詳情視窗。您可以在此更新所有任務屬性、新增備注、查看子任務，以及查看完整的任務歷史記錄。需要更徹底的編輯或需要上下文時請使用此選項。
  </Tab>
</Tabs>

## 任務刪除規則

「所有任務」頁面上的任務直接與其上層訂單相關聯。

- 當 **訂單被刪除** 時，與該訂單關聯的所有任務將從「所有任務」頁面 **自動永久移除**。
- 此刪除影響所有使用者角色——訂單刪除後，任何人都將無法看到這些任務。
- 透過訂單刪除而刪除的任務 **沒有還原選項**。

<Warning>
  刪除訂單會永久移除其所有任務。在刪除訂單之前，請確保任何重要的任務歷史記錄或可交付成果記錄已另存他處。
</Warning>

## 附加說明

<Accordion title="即時更新">
  對訂單、任務指派或任務狀態的變更會即時反映在「所有任務」頁面上。如果隊友在您開啟頁面時更新了任務，您無需重新整理即可看到變更。
</Accordion>

<Accordion title="客戶請求任務">
  客戶提交的任務在代理商明確接受之前不會出現在「所有任務」頁面上。在接受之前，這些任務保持待處理狀態，在集中檢視中不可見。
</Accordion>

<Accordion title="任務與訂單狀態">
  任務只對進行中或審核中狀態的訂單顯示。如果訂單移至已完成或暫停狀態，其任務將不再顯示在此處。若要查看這些狀態以外的訂單所關聯的任務，請前往特定訂單。
</Accordion>
