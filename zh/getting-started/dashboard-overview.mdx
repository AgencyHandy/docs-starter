登录后，AgencyHandy 将带您进入主仪表板——这是管理代理机构一切事务的控制中心。界面简洁，以导航为驱动：持久显示的左侧边栏可快速访问所有主要功能，而中央面板会随您选择的区域而更新，显示相关数据。运营代理机构所需的一切——从客户关系到发票，再到任务跟踪——都在这里。

## 两种仪表板视图

AgencyHandy 根据登录者的身份提供两种不同的仪表板视角，您可以按需切换。

<CardGroup cols={2}>
  <Card title="代理机构仪表板" icon="building">
    代理机构所有者、管理员和团队成员的默认视图。使用它可实时了解代理机构的运营状况——活跃订单、待处理工单、未付发票、任务进度和团队工作量——一切尽在一个中央视图中。这是您将花费大部分时间的仪表板。
  </Card>
  <Card title="客户仪表板" icon="user">
    您的客户登录 AgencyHandy 时访问的专属门户。客户可以监控其活跃订单、查看分配的任务、检查未付发票并提交支持工单——让他们对您代为交付的工作拥有完全的透明度。
  </Card>
</CardGroup>

<Note>
  代理机构仪表板和客户仪表板显示相同的底层数据，但视角不同，权限级别也不同。客户只能看到与其自身账户相关的数据；代理机构用户可以看到所有客户的数据。
</Note>

## 左侧边栏导航

左侧边栏是您的主要导航面板。每个图标和标签对应 AgencyHandy 的一个主要功能区。以下是各区块的功能说明：

### CRM
您的客户关系管理中枢。存储和管理所有客户联系方式，跟踪沟通历史，并维护代理机构合作的每个账户的清晰记录。将 CRM 作为了解客户是谁以及每段关系现状的唯一可信来源。

### 服务
定义代理机构提供的服务并构建服务目录。客户下单时，将从您在此处发布的服务中进行选择。您可以为每项服务设置定价、描述、交付成果和订单接收表单。

### 订单
从下单到完成，跟踪每一个客户订单。订单区块可让您了解订单状态、关联任务、客户沟通和交付时间线——确保销售与交付之间不出现任何疏漏。

### 提案
直接在 AgencyHandy 中创建、发送和管理提案及合同。起草提案，附上定价和工作范围，然后发送给客户审阅。一旦接受，提案可转换为活跃订单，无需任何手动重复操作。

### 发票
生成与客户订单或项目关联的专业发票，跟踪付款状态，并让客户清楚了解应付款项。AgencyHandy 处理从创建到结算的完整发票生命周期，减少账单往来中通常存在的来回沟通。

### 任务
在任务层面管理所有工作。创建任务，分配给团队成员，设置截止日期和优先级，并监控完成状态。任务直接关联至订单和服务，因此进度始终在上下文中可见。

### 团队
在集中位置管理您的团队成员。邀请新成员、分配角色，并控制每个人在工作区中的访问权限。随着代理机构发展，使用此区块保持团队名册的最新状态。

### 工单
通过内置工单系统处理客户支持请求和内部问题跟踪。客户可以从其门户提交工单，您的团队无需切换至独立的服务台工具即可对其进行分类、回复和解决。

### 工时表
跟踪客户工作和内部项目的时间投入。团队成员可以按特定任务或订单记录工时，为您提供准确的计费、产能规划和项目盈利能力报告数据。

### 表单
构建和管理自定义表单，在客户引导、订单接收或工作流中的任何其他节点收集客户的结构化信息。在此创建的表单可附加到服务、提案，或直接发送给客户。

### 文件
在一个集中位置存储和整理与代理机构工作相关的所有文件——交付成果、合同、品牌资产和客户文档。文件可按特定客户或订单归类，让所有人随时知道在哪里找到所需内容。

### 设置
配置 AgencyHandy 工作区的每个方面。管理代理机构档案、品牌、通知偏好、集成、团队权限、账单等。设置也是查找适用于所有客户和团队成员的工作区级别配置的地方。

## 快速参考：边栏区块一览

| 区块 | 主要用途 |
|---|---|
| **CRM** | 管理客户联系方式和账户记录 |
| **服务** | 定义和发布代理机构的服务目录 |
| **订单** | 端到端跟踪和交付客户订单 |
| **提案** | 创建、发送和管理提案及合同 |
| **发票** | 生成发票并监控付款状态 |
| **任务** | 分配工作、设置截止日期并跟踪完成情况 |
| **团队** | 邀请成员、分配角色、控制访问权限 |
| **工单** | 处理客户支持请求和内部问题 |
| **工时表** | 记录和查看工作时间 |
| **表单** | 构建客户数据收集的自定义表单 |
| **文件** | 集中存储文档和交付成果 |
| **设置** | 配置工作区、品牌和权限 |
