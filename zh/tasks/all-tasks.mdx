AgencyHandy 中的所有任务页面为您提供了一个单一的集中视图，显示团队中所有活跃工作流中的每项任务。您无需逐一翻查各个订单，而是可以在一个地方查看所有相关任务，按负责人、优先级或日期进行筛选，并直接在列表中进行编辑。可见性会根据您的角色自动限定范围——超级管理员可以看到所有内容，项目经理只能看到其分配的订单，员工只能看到自己的任务。

## 访问所有任务页面

1. 登录您的 AgencyHandy 账户。
2. 点击左侧边栏中的 **任务**。
3. 选择 **所有任务** 打开集中任务工作台。

<Note>
  所有任务页面仅显示处于 **进行中** 或 **审核中** 状态的订单中的任务。与处于其他状态的订单关联的任务（例如已完成、待处理）不会在此显示。
</Note>

## 按角色的任务可见性

您在所有任务页面上看到的内容完全取决于您的角色。

| 角色 | 可见内容 |
|------|---------------|
| **超级管理员/管理员** | 所有订单、团队和状态下的所有任务 |
| **项目经理（PM）** | 仅限他们被指定为负责人的订单中的任务 |
| **员工/负责人** | 仅限直接分配给他们的任务——即使队友正在处理同一个订单 |

<Tip>
  如果项目经理找不到他们预期看到的任务，请确认他们已被指定为相关订单的负责人。项目经理只有在被指定为该订单的负责人时才能看到任务。
</Tip>

## 任务何时出现在此页面

当满足以下条件时，任务将显示在所有任务页面：

<CardGroup cols={2}>
  <Card title="触发任务可见性的订单" icon="circle-check">
    - 创建了手动订单（一次性或订阅）
    - 通过 Stripe 或手动付款方式下达的目录订单
    - 提案被团队接受
  </Card>
  <Card title="保持任务隐藏的订单" icon="eye-slash">
    - 团队尚未接受的客户请求任务
    - 团队尚未接受的试用订单
    - 等待手动银行付款确认的待处理状态订单
  </Card>
</CardGroup>

## 搜索、筛选和排序

所有任务页面提供了一套完整的控件，帮助您精确找到所需内容。

### 搜索

使用搜索栏按名称、负责人或相关订单查找任务。结果会在您输入时实时更新。

### 筛选

点击 **筛选 → + 添加筛选** 打开筛选面板。您可以组合多个筛选条件以获得精确结果：

| 筛选条件 | 作用 |
|--------|--------------|
| **订单 ID** | 显示关联到特定订单编号的任务 |
| **负责人** | 显示分配给特定团队成员的任务 |
| **状态** | 按任务进度筛选：待处理、进行中、审核中、已完成等 |
| **优先级** | 聚焦于高、中或低优先级任务 |
| **标签** | 按任务标签或分类缩小范围 |
| **启动日期** | 显示在特定日期当天或之后开始的任务 |
| **结束日期** | 显示在特定日期前到期的任务 |

<Note>
  筛选条件仅在所有任务页面上可用。它们在您的浏览器会话期间保持有效，但在页面刷新时会重置。筛选条件只影响您的视图，不会改变其他用户看到的内容。
</Note>

要清除所有筛选条件，点击 **清除筛选** 以返回完整的未过滤列表。

### 排序

按以下条件对任务列表排序：
- **截止日期** — 将即将到来或已逾期的任务显示在顶部
- **优先级** — 将最关键的任务置于顶部
- **创建时间** — 最新添加的任务优先显示

### 面包屑导航

使用页面顶部的面包屑路径，在导航子任务、任务列表和父级订单时保持上下文。当您深入查看特定任务后需要返回更广泛的列表时，这特别有用。

## 编辑任务

您可以通过两种方式在所有任务页面上修改任务：

<Tabs>
  <Tab title="行内编辑">
    直接点击任务行中的可编辑字段——例如状态、负责人或截止日期——无需离开列表视图即可进行更改。这是快速更新的最快方式。
  </Tab>
  <Tab title="弹窗编辑">
    点击任务名称以打开完整的任务详情弹窗。在这里您可以更新所有任务属性、添加备注、查看子任务，以及查看完整的任务历史。需要进行更全面的编辑或需要上下文时，请使用此方式。
  </Tab>
</Tabs>

## 任务删除规则

所有任务页面上的任务直接与其父级订单关联。

- 当 **订单被删除** 时，与该订单关联的所有任务都会从所有任务页面 **自动且永久地移除**。
- 此删除影响所有用户角色——订单删除后，任何人都无法看到这些任务。
- 通过订单删除而删除的任务 **没有恢复选项**。

<Warning>
  删除订单会永久移除其所有任务。在删除订单之前，请确保将任何重要的任务历史或可交付成果记录保存到其他地方。
</Warning>

## 附加说明

<Accordion title="实时更新">
  订单、任务分配或任务状态的变更会在所有任务页面上实时反映。如果队友在您打开页面时更新了任务，您将无需刷新即可看到更改。
</Accordion>

<Accordion title="客户请求的任务">
  客户提交的任务在团队明确接受之前不会显示在所有任务页面上。在被接受之前，这些任务处于待处理状态，在集中视图中不可见。
</Accordion>

<Accordion title="任务与订单状态">
  任务仅针对处于进行中或审核中状态的订单显示。如果订单变为已完成或被暂停，其任务将不再在此显示。请导航至具体订单以查看与这些状态之外的订单关联的任务。
</Accordion>
